关于“四清”和“文革”交叉处理的案件，经复查改正后工资补发问题的通知

各地、市委组织部，区直各单位党组（党委）：

“四清”和“文化大革命”交叉处理的案件，经复查改正后的工资补发问题，区党委纪检会一九八○年九月发的桂纪字[1980]44号文件已作了明确的规定，各单位都按此文件精神执行了。但近几年来，。还有一些人申诉，为了处理好这一遗留问题，再作如下的通知：

“四清”和“文化大革命”交叉处理的案件，经复查改正后工资补发问题，应继续执行区纪检会桂纪发[1980]44号文件，但属于下列情况的可按“文革”案件，工资予以补发。

1、“四清”运动已查清定案和作出处理决定，由于各种原因没有宣布执行。直到当地革委会或单位革命领导小组成立后才宣布处理的口现经复查，属于冤假错案，被降减的工资，应作为“文化大革命”的案件予以补发。

2、“四清”运动中，“四清”工作团立案审查，提出拟作法办的案件，由于各种原因，当时法院没有审理判决，而在当地革委会成立之后，法院才审理判决的，经复查属冤假错案，应作为“文化大革命”的案件，被扣发或停发的工资，应予补发。

中共广西壮族自治区委员会组织部一九八三年五月二十三日

来源：

( 《处理“文革”遗留问题、清理“三种人”文件汇编》，广西壮族自治区党委整党办公室，一九八六年八月。)
