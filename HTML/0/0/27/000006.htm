关于广西壮族自治区整党办“处遗”组撤销后业务归口问题的通知

各地、市、县委，柳铁党委，防城港区工委，区直各单位党组（党委）：

我区处理“文革”遗留问题工作，自一九八三年以来，在各级党委的共同努力下，取得了很大的成绩。但是，“处遗”中还有一些遗留问题需要继续抓紧解决，以巩固和发展“处遗”成果，维护全区安定团结的政治局面。目前，全区整党工作已基本结束，区整党办公室“处遗”组随之撤销，今后，“处遗”工作中的遗留问题归口有关业务部门办理。特作如下通知：

一、有关桂发[1983]55号及桂政发[1984]133号文件规定的遗属子女就业安置工作等遗留问题，统一归口由区劳动局办理。

二、有关桂政发[1985]126号文件规定的“文革”中被辞退的代课老师收回等遗留问题，归口由区教育厅、区劳动局办理。

三、有关“处遗”经费的开支问题，按区党委组织部、区党委整党办公室、区财政厅桂组通字[1987]2号文件的规定办理。

四、关于“处遗”的来信来访，归口由区信访部门办理。

五、有关未结案的“处遗”案件，转核查“三种人”小组办理。

六、有关“处遗”，档案，由秘书组统一向区党委机关档案室移交归档。

中共广西壮族自治区委员会办公厅中共广西壮族自治区委员会整党办公室一九八七年七月九日

来源：

( 《处理“文革”遗留问题、清理“三种人”文件汇编》，广西壮族自治区党委整党办公室，一九八六年八月。)
