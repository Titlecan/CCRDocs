关于到区档案馆查用“文革”中专案材料的事项的通知

日期：1984-1-24 作者：[待确定]

各地、市、县委，柳铁党委，区直各单位党组（党委）：

近年来，由于“处遗”、审干和选拔干部的需要，不少单位派人到区档案馆要求查用“文化大革命”中的专案材料。现根据中共中央办公厅（1983）136号文件规定的精神，对这部份档案资料的使用，作如下规定：

一、各地、各部门需要到区档案馆查用“文革”中专案材料，必须由各地、市、县委、区直机关党组（党委）以及县以上企事业单位党委向区党委办公厅写查档公函，并由负责人签字。查档公函应注明查阅人员的身份，使用档案的目的、内容和要求（查阅、借阅、摘抄、复制），经区党委办公厅批准后区档案馆方予办理。

二、到区档案馆查用“文革”中档案资料的人员，必须是政治上绝对可靠，组织性、纪律性和保密观念强的同志，查阅范围不能扩大，材料内容不得扩散。

三、在“文革”期间的文书档案中，凡涉及个人问题的材料，不应作为考查干部的依据。考查干部的历史应以人事档案为准。

四、未经同意，档案一律不得摘抄、复印。经过同意摘抄，复印带走的材料，必须严格登记，办理签收手续，妥善保管，其中特别机密的部份，用完后要按照区档案馆的规定退回。

中共广西壮族自治区委员会办公厅一九八四年一月二十四日

来源：

( 《处理“文革”遗留问题、清理“三种人”文件汇编》，广西壮族自治区党委整党办公室，一九八六年八月。)
