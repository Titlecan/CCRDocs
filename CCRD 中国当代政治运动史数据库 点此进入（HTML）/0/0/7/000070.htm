中央军委关于学习五十四军做群众组织工作的通知

日期：1967-5-27 作者：[待确定]

各军区，各军种，各兵种：

54军在重庆的工作做得不错，可供你们参考。凡属两派革命群众组织互相对立严重的，则要进行细致的思想政治工作，问题总是可以解决的，对于有群众的保守组织，甚至反动组织，也要做细致的思想政治工作。

一九六七年五月二十七日

来源：

根据《中共中央文件汇编，1966.5-1968.5 [内部参考]》，1968年5月出版，编印者不明
