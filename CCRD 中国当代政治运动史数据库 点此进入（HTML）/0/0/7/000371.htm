中央文革“2·24”对西藏文革的指示

日期：1967-2-24 作者：[待确定]

拉萨革命造反总部、拉萨革命造反公社、首都赴藏革命造反总部、首都三司驻藏联络站、北航“红旗”小分队等各民族革命造反派：

我们收到你们2月22日从拉萨发来的电报。对你们提出的问题，答复如下：

一、中央军委2月12日给西藏军区的电报，是代表毛主席、党中央的意见的。中央文化革命小组完全拥护这个电报所说的：张国华同志是站在毛主席路线一边的，尽管他在工作上有缺点和错误，但基本上是个好同志。望军区党委同志向革命群众做些工作。

二、中央文化革命小组，欢迎你们要掌握斗争大方向，捍卫和实现伟大领袖毛主席提出的革命的“三结合”联合夺权的伟大方针。你们要同张国华同志和西藏军区革命领导同志合作，使“西藏地区的无产阶级文化大革命不走弯路”。

此致

无产阶级文化大革命的敬礼

中央文化革命小组1967年2月24日

来源：《中央文件汇编》，拉萨革命造反总部，1969年3月。
