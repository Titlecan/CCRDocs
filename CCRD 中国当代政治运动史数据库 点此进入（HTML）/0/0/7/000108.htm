全军文革办传达中央文革小组关于湖南“湘江风雷”、“红旗军”的批示

日期：1967-2-4 作者：[待确定]

湖南军区对“湘江风雷”和“红旗军”的头目，应当立即采取专政措施，分化瓦解其中受蒙蔽的群众。这个意见并电告广州军区。

全军文革办一九六七年二月四日七时

来源：

原载湖南《反复辟战报》1967年7月17日，第一期，第一版
