中央军委给总参三部和技术工程学院的指示

三部全体同志们和技术工程学院的同学们，军委一月九日召开了会议，中央文革小组陈伯达、江青同志参加了这个会议，军委考虑到三部所担负的战备任务极为繁重，决定三部机关的文化大革命暂时不进行。三部什么时候进行文化大革命，军委将根据以后的情况再决定。

中央军委一九六七年一月九日
