全军文革小组给河南军区的电报

河南军区转总字706部队总字344部队等进驻河南军区办公楼的革命群众组织的同志们：

你们要立即撤出河南军区办公室，不要冲击军事机关。河南的问题，中央召集各方代表来京开会，会正确解决的，你们和河南的所有军事院校的同志们，都回到本校去搞本单位的斗批改。

中央军委文革小组一九六七年五月二十八日

来源：

原载北京化工学院毛泽东思想宣传员 《无产阶级文化大革命文件汇编 [第二集]》，北京：1967年9月
