中指委办公室转发杭州利民制药厂《关于“三种人”黄信川在被处理前后的活动情况》

按：近来一些地方反映，有的“三种人”在结论处理后，气焰嚣张，甚至继续拉帮结派，危害党和人民的事业；有的还扬言什么：“政治上不行了，在经济上要捞一把”。杭州市利民制药厂原厂长、“三种人”黄信川就是一个例证。

杭州市医药公司党委对黄信川的处理是完全正确的。但黄信川这样的人却拒不改悔，从根本上说，是他们坚持林彪、“四人帮”的反动思想体系的反映。从我们一些地方和单位的工作来看，同教育管理不严、听之任之也有关系。对“文革”中犯了“三种人”错误或罪行以及犯了严重错误的人，按照党的政策做出结论给以处理，这是为了严肃党的纪律，纯洁组织，也是为了对犯错误的人进行教育、挽救。因此，在对这些人结论处理之后，各级党的组织，特别是组织、人事部门要加强教育管理，不能自由流动，并有针对性地做好思想政治工作，促使他们向积极方面转化。对他们中有悔改表现的应予鼓励，对坚持错误的要批评教育，对继续干坏事触犯纪律和法律的要从严处理。

杭州利民制药厂原厂长黄信川，因在“文化大革命”中犯有严重的错误和罪行，市医药公司党委于八四年八月十八日决定将他调离现职，责令其交待问题，并于十二月十三日决定把他定为“三种人”中的“打砸抢”分子，撤销厂长职务，开除党籍。

黄信川自去年三月市医药公司党委开始对其审查以来，秘密串连，订立攻守同盟，拒不交待自己的问题。在去年八月十八日公司对他采取调离现职，交待问题的组织措施后，当天晚上，黄叫了厂内几个干部和工人到家中商量对策。他说：“我是没有什么问题的，今后我也不会回厂了，我准备搞一个‘开发公司’，你们出来和我一起干。”他多次动员销售员李涛，科室人员徐长江等和他一块搞公司。他还多次在自己家中请客，声称“三个月内打回利民厂”。同时，’他利用个别与他关系密切的人在厂内散布舆论，说“黄信川是暂时调离厂”，“现在是一朝天子一朝臣！”他们还采用写纸条的手法，攻击不同意他们作法的老工程师陈国宾是“忘恩负义，小人！”。黄信川在与新任厂长姚祖丰同志交接工作时说：“如果你要搞我，那我也会搞你的，当然你现在没有搞我，我也没有搞你。”去年十二月十二日，厂党支部召开全厂党员大会，讨论对黄信川的党纪处分问题。会前，黄同党支部书记王成纲、厂长姚祖丰讲：“最近福建樟州香料厂的厂长到我家来说，‘樟州市经委和他们厂领导集体讨论决定，不管你有什么问题，不管给什么处理，我们都聘请你到樟州去。”

去年十一月以来，在黄信川的唆使拉拢下，该厂见习副厂长兼计划科长宗泽后，采购员陈申炎，销售员徐向荣，厂长助理员兼办公室主任宋俐俐先后要求辞职，在未经批准的情况下不告而别，擅自离厂。宗泽后还将该厂生产的风油精的技术工艺和配方带往温州文成制药厂，为这个厂生产风油精提供资料。宗泽后、陈申炎、徐向荣、宋俐俐还利用他们在利民厂的业务关系，四出活动，以每瓶风油精一分钱的经销费用现金支付对方个人手中，致使十多个业务单位废除了与利民厂已签订的供货合同。同时他们将风油精的工艺配方提供给温岭日用化工厂。利民厂去年研制的“麒麟牌”风痛灵，是该厂的第二个拳头产品，尚未大批量生产。去年十二月上旬，文成制药厂上报浙江省医药总公司，要求批准生产风痛灵，其工艺，配方均与利民厂一样。此产品的工艺配方极为保密，只有黄信川、宗泽后和技术部门才有。经查，宗泽后原保存的一套资料离厂时带走，未作任何移交，导致了该产品的工艺配方的扩散。

今年三月二十八日，黄信川因有流氓活动和经济问题，被杭州市下城区公安机关收容审查，并对其家中进行搜查。发现黄信川自去年十月份以来，先后被文成制药厂、文成工业供销公司聘为“顾问”，各领发了一本“工作证”，文成工业供销公司给他每月工资一百元，另加一年一百二十元的书报费。黄家中还存有上述两个单位的业务公章各一枚，空白介绍信二本，空白合同书五本。黄信川等人为这两个单位推销了一百二十多万瓶风油精，共领取一万一千多元业务费。据黄本人交待，他曾经将风痛灵资料“零星地”给了文成制药厂。另外从黄家中查出其停职交待问题期间在家中召开准备建立“杭州市医药工业供销公司”的会议记录。
