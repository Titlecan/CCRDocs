国务院关于停办农垦学校问题给安徽省军管会的批复

日期：1968-3-18 作者：[待确定]

一九六八年二月十二日报告收悉。关于停办你省过去用安置经费办的农垦学校的问题，仍请省军管会负责继续处理。坚持知识青年下乡上山的正确方向，动员各校现有学生到农村社队落户，一定要高举毛泽东思想伟大红旗，突出无产阶级政治，以斗私批修为纲，充分做好思想政治工作。所需安置经费可列入今年年度安置计划指标内，不必专案另报。一定要坚决贯彻执行伟大领袖毛主席“要进一步节约闹革命”的指示，充分利用各校现有的生产生活资料，能不花钱的事就不花钱，依靠群众，坚持发扬我党艰苦奋斗的光荣传统。
