广西壮族自治区革命委员会筹备小组、中国人民解放军广西军区关于破获反革命集团“中华民国反共救国团广西分团”反革命案件的公告

“千钧霹雳开新宇，万里东风扫残云”。

在伟大领袖毛主席革命路线的指引下，无产阶级文化大革命的形势一片大好，越来越好。规模空前的革命群众运动，象大海的怒涛冲垮了以中国赫鲁晓夫为首的资产阶级司令部，把那些叛徒、特务、死不悔改的走资派，以及国民党反动派的残渣余孽冲刷出来，粉碎了他们妄图颠覆无产阶级专政，复辟资本主义，恢复半封建、半殖民地制度的反革命罪恶阴谋。

最近，我区无产阶级革命派、广大革命人民群众，和专政机关共同破获了反革命集团“中华民国反共救国团广西分团”的反革命案件，给了那些面临末日的一小撮阶级敌人又一次沉重的打击。这是战无不胜的毛泽东思想的伟大胜利，是毛主席革命路线的伟大胜利！

蒋匪“中华民国反共救国团广西分团”，代号“57125”，是一个以国民党反动派的残渣余孽和没有改造好的地、富、反、坏、右分子为其主要成员，以破坏无产阶级文化大革命，阴谋颠覆无产阶级专政，恢复国民党反动统治为其罪恶目的的反革命集团。

这个反革命集团的主要骨干，在无产阶级文化大革命开始以后，就相继集中南宁，乘机四出活动，发展反革命势力。今年二月二十三日，由国民党特务、伪宪兵朱锡生、坏分子梁汉忠、反革命分子韦国书、伪自卫中队长、劳动释放犯杨德标、坚持反动阶级立场的地主子弟覃德玮、盗窃犯农培相，等等，在南宁召开反革命骨干分子会议，正式成立蒋匪“中华民国反共救国团广西分团”；制定并通过了反革命纲领和组织章程；由梁汉忠任团长，杨德标、韦国书等任副团长；分团、政治部、作战部、情报部、经济部、外事处，下辖纵队（支队）、大队、中队、分队、小队。分团部设在南宁市××路××街××号。

这个反革命集团的政治纲领极端反动。他们声称其总后台是人民公敌蒋介石，总团长是匪首蒋经国。他们公然叫嚣要推翻共产党，推翻社会主义制度，破坏中国人民解放军，破坏新生的红色政权——革命委员会，颠覆无产阶级专政，恢复国民反动统治。他们目前的反革命行动计划是：加紧反革命宣传，网罗牛鬼蛇神，发展反革命组织，结成反革命的“统一战线”，“在城市和农村”建立反革命“根据区域”，“储蓄力量，暂时埋伏，以待时机”，“在条件成熟时，以农村为中心，以山区为重点”，发展反革命武装，组织反革命暴乱，梦想配合国民党反动派“反攻大陆”。

这个反革命集团有计划地派出一些骨干分子，有的借“革命”串连之名，有的以“行医”、“经商”作掩护，四出活动，先后到柳州、玉林、钦州、灵山、浦北、宁明、横县、邕宁、德保、靖西、来宾等县、市，网罗牛鬼蛇神，发展反革命组织；并到广东、江西等省，与当地反革命组织勾结，进行反革命活动。同时，他们还从外地流窜来南宁的坏人中物色对象，发展组织。他们出版了《太阳报》、《灯塔》报等反动刊物，刊载蒋匪帮的反革命宣传资料，进行反革命宣传活动。

这个反革命集团的一些骨干分子，混入某些群众组织，有的还当了坏头头，打着“造反”旗号，狡猾地利用群众组织掩护他们进行反革命活动。在匪骨干会议上，匪“广西分团”政治部副部长梁孟隆布置：要“插手两派斗争，掌握×××（某群众组织）的领导权，利用矛盾，挑起武斗，夺取武器，实行武装夺权。”分团长梁汉忠叫嚣：“要拖住革命委员会的成立，破坏革命委员会的成立，以便有利于‘反共救国团’的发展。”他们利用群众组织中某些人的资产阶级派性，或者直接操纵一些人，或者在背后煽阴风，点鬼火，挑动武斗，制造混乱，破坏革命大联合和革命三结合。他们从某些群众组织中骗取枪支、弹药和爆炸物品，并大搞打、砸、抢、抄，越境走私，攫取反革命活动经费，建立反革命武装。主犯覃德玮、农培相等人，通过南宁市某些群众组织据点中的涂水成，取得了该群众组织的出入证，并在该据点接待、联络这个反革命集团的外地成员。主犯梁汉忠、朱锡生、廖钧琅、梁孟隆、黄廷敏等，在革命群众协同专政机关追捕时，有的从某些群众组织弄来的枪支开枪拒捕，有的逃到某些群众组织的据点隐藏起来。

无产阶级革命派的战友们，革命的同志们：

我们这里公布的只是这个反革命集团的一部分罪恶事实。在这些事实中，我们可以清楚地看到，一小撮阶级敌人“每日每时企图在中国复辟”，“如果我们丧失警惕性，那就会上大当，吃大亏。”现在这个反革命集团并没有完全摧毁，必须彻底追查组织，缉拿漏网的主犯归案。

伟大领袖毛主席教导我们：“反革命分子怎样耍两面派手法呢？他们怎样以假象欺骗我们，而在暗里却干着我们意料不到的事情呢？这一切，成千成万的善良人是不知道的。就是因为这个原故，许多反革命分子钻进我们的队伍中来了。我们的人眼睛不亮，不善于辨别好人和坏人。我们善于辨别在正常情况之下从事活动的好人和坏人，但是我们不善于辨别在特殊情况下从事活动的某些人们。”事情正是这样，当我们对阶级敌人的破坏麻痹大意的时候，特别是当我们被资产阶级派性蒙住了眼睛，分不清敌我，混淆两类不同性质的矛盾，把矛头对着自己的阶级兄弟，在革命群众组织之间大打“内战”的时候，阶级敌人的黑手就伸进来了。从蒋匪“反共救国团”的反革命案件中，我们应当很好吸取这一教训，学得更聪明一些。还应当看到，在这一反革命案件公布之后，一小撮反革命分子决不会就此甘休，束手就擒的。他们必定还会以种种手法，进行挑拨离间，制造事端，兴风作浪。对此，我们决不能掉以轻心，再上敌人的当。

无产阶级革命派和广大革命群众，必须更高地举起毛泽东思想伟大红旗，紧跟毛主席的伟大战略部署，牢牢掌握革命斗争大方向，牢记毛主席“千万不要忘记阶级斗争”的伟大教导，坚决贯彻执行毛主席的最新指示，增强阶级斗争观念，擦亮眼睛，提高革命警惕性，进一步克服资产阶级派性，增强无产阶级党性，认真清理阶级队伍，纯洁组织，团结一致，共同对敌，严格区分敌我矛盾和人民内部矛盾，团结一切可以团结的力量，坚决保护广大人民群众，组织起浩浩荡荡的革命大军，主动地、不停顿地向阶级敌人发动全面的、猛烈的进攻，彻底摧毁反革命集团“中华民国反共救国团广西分团”，并乘胜追击，开展大揭发、大检举、大斗争，把隐藏的一切反革命分子统统挖出来！我们希望，通过这场斗争，能够更好地促进、巩固和发展革命的大联合和革命的三结合，推动革命大批判更深入地、持久地开展起来，夺取无产阶级文化大革命的全面胜利。

参加反革命集团“中华民国反共救国团广西分团”的所有成员和一切反革命分子，摆在你们面前的唯一出路是向人民投案自首，低头认罪。我们在这里向你们宣布：“坦白从宽，抗拒从严。”如能主动投案坦白者，从宽处理，并能主动检举其他反革命分子者，将功赎罪；如果顽抗到底，必将被无产阶级专政的铁拳砸得粉身碎骨。
