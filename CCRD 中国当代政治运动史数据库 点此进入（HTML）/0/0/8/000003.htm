商业部关于拨给下乡青年补助用布问题给天津市革委的复函

你委给国务院关于拨给下乡青年补助棉布、棉花的请示报告，国务院批由我部和供销总社办复。关于棉布问题，我部同意根据国务院批准到内蒙古农村插队落户的初、高中毕业生人数，按每人六十尺的标准拨给冬装补助用布。具体事项请通知天津市纺织品公司办理。
