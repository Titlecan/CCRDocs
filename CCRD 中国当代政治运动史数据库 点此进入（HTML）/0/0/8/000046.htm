中共西藏军区委员会关于成立西藏自治区革命委员会的请示报告

日期：1968-8-28 作者：中共西藏军区委员会

中共西藏军区委员会

毛主席、林副主席、中共中央、国务院、中央军委、中央文革：

在全国亿万军民夺取无产阶级文化大革命全面胜利的凯歌声中，我们怀着万分激动的心情，代表西藏地区无产阶级革命派、各族革命人民、红卫兵革命小将和人民解放军驻藏部队，向我们最敬爱的伟大领袖毛主席和他的亲密战友林副主席报喜：成立西藏自治区革命委员会的筹备工作已经就绪。这是毛泽东思想和毛主席革命路线的伟大胜利。我们衷心祝愿伟大领袖、我们心中最红最红的红太阳毛主席万寿无疆！万寿无疆！

（一）

西藏地处我们伟大祖国的西南方，是我国反对美帝、反对苏修、反对印度反动派斗争的前哨，是祖国大陆最后解放的一个边疆少数民族地区。伟大领袖毛主席教导我们：“民族斗争，说到底，是一个阶级斗争问题。”西藏和平解放17年来，两个阶级、两条道路、两条路线的斗争，一直十分尖锐、复杂。西藏各族革命人民和驻藏人民解放军，在伟大统帅毛主席的英明领导下，同国内外阶级敌人进行了长期、激烈、英勇的斗争，经过反复较量，驱逐了帝国主义势力出西藏，推翻了反动的农牧主阶级统治，彻底粉碎了以达赖、班禅为代表的反动农奴主阶级勾结美、英帝国主义、苏联修正主义、蒋介石匪帮和印度反动派，分裂祖国统一，反对社会主义，妄图复辟封建农奴制度的罪恶阴谋；彻底粉碎了彭德怀、高岗、习仲勋反党集团黑干将范明的三反滔天罪行，取得了一个又一个的重大胜利。

伟大领袖毛主席教导我们：“无产阶级文化大革命，实质上是在社会主义条件下，无产阶级反对资产阶级和一切剥削阶级的政治大革命，是中国共产党及其领导下的广大革命人民群众和国民党反动派长期斗争的继续，是无产阶级和资产阶级阶级斗争的继续。”党内最大的一小撮走资本主义道路的当权派刘少奇、邓小平、陶铸、彭德怀、贺龙、彭真、罗瑞卿、陆定一、杨尚昆、谭震林、薄一波、安子文、习仲勋、李井泉及其在西藏的代理人周仁山、王其梅等长期以来，代表国民党反动派和农奴主阶级的利益，疯狂反对我们伟大领袖毛主席，反对战无不胜的毛泽东思想，反对社会主义。他们招降纳叛，结党营私，勾结达赖、班禅反动集团和国民党的残渣余孽，在西藏实行阶级投降主义，推行一条反革命修正主义路线，妄图复辟封建农奴制度和资本主义。这一小撮反革命修正主义分子，是西藏各族革命人民的死敌。在无产阶级文化大革命运动中，他们感到末日来临，便顽固地推行资产阶级反动路线，大耍反革命两面手法，颠倒是非，混淆黑白，挑动群众斗群众，拼死转移斗争大方向，制造白色恐怖，把革命群众打成“反革命”，充分暴露了他们反革命狰狞面目。

伟大领袖毛主席教导我们：“人民，只有人民，才是创造世界历史的动力。”西藏第一代工人阶级，各族革命人民、红卫兵革命小将，怀着誓死保卫毛主席，誓死保卫毛泽东思想，誓死保卫毛主席革命路线的深厚的无产阶级感情，在毛主席《炮打司令部》的伟大号召下，高举“对反动派造反有理”的革命大旗，向党内最大的一小撮走资派及其在西藏的代理人周仁山、王其梅等，发起猛攻。他们发扬了无产阶级的革命英雄气概，不怕打击，不怕迫害，勇往直前，冲破重重阻碍，粉碎了阶级敌人一次又一次反扑，终于把中国赫鲁晓夫在西藏的代理人周仁山、王其梅等一小撮叛徒、特务、死不悔改的走资派揪了出来，使他们罪恶的阴谋遭到了可耻的失败，夺取了西藏地区无产阶级文化大革命决定性胜利。他们为西藏无产阶级文化大革命立下了不朽的功勋。

（二）

伟大领袖毛主席和他的亲密战友林副主席、中央、中央文革，对西藏地区无产阶级文化大革命极为关怀，在两个阶级、两条道路、两条路线斗争的每一关键时刻，都及时地给我们作了英明指示，为我们指明了前进的方向。毛主席的每一条最新指示，都是我们的指路明灯，都是推动西藏文化大革命运动前进的最强大动力。

去年9月18日和今年6月6日，在全国无产阶级文化大革命不断深入发展的大好形势下，中央、中央文革负责同志两次接见了军区党委在京常委，对西藏无产阶级文化大革命作了极为重要的指示，是对我们深刻的教育，给我们指出了解决西藏文化大革命问题的根本方向。今年6月22日，中央文革对《关于立即停止西藏地区武斗的协议》的批示，8月17日，中央、中央文革负责同志接见在京学习的师以上干部代表，8月26日，又接见了西藏革命群众组织代表，再次给我们作了极为重要的指示。特别是5月8日和8月11日，伟大领袖毛主席和他的亲密战友林副主席亲切接见了西藏在京学习的两个学习班的全体同志，这是对我们的最大信任，最大关怀，最大鼓舞，也是我区军民的最大光荣，最大幸福。这就大大地激发了全区军民无限忠于毛主席，无限忠于毛泽东思想，无限忠于毛主席革命路线的深厚的无产阶级感情，极大地推动了西藏革命大联合的发展和革命委员会的成立。毛主席最近指出：“我国有7亿人口，工人阶级是领导阶级。要充分发挥工人阶级在文化大革命中和一切工作中的领导作用。工人阶级也应当在斗争中不断提高自己的政治觉悟。”毛主席这一最新指示，极为重要，对于我们完成面临的各项任务有重大的意义。毛主席批准的“七·三”布告和“七·二四”布告，对于我们的情况，同样完全适用，必须严格遵守。毛主席所有这一切指示，都深刻地说明，西藏无产阶级文化大革命的每一个胜利，都是战无不胜的毛泽东思想的胜利，都是毛主席的无产阶级革命路线的胜利，让我们千遍万遍高呼：毛主席万岁！万岁！万万岁！

人民解放军驻藏部队广大指战员积极响应伟大统帅毛主席发出的“人民解放军应该支持左派广大群众”的伟大号召，在执行“三支”“两军”光荣任务中，做了大量工作，取得了巨大成就，但在支左工作中，军区党委常委犯了很多错误，主要是支一派、压一派，亲一派、疏一派，在有的问题上犯了方向路线错误。有了错误就改，这次军区党委常委在西藏学习班中做了检讨。这就使我们在阶级斗争中经受了考验和锻炼，进一步提高了阶级觉悟和路线斗争觉悟。

随着无产阶级文化大革命运动的开展，在西藏我们广泛地开展了“拥军爱民”活动，加强了军民团结，粉碎了国内外阶级敌人的阴谋和破坏，打退了印度反动派的武装挑衅和叛匪的回窜骚扰，保卫了无产阶级文化大革命，保卫了祖国西南边防。同西藏各族人民生死与共，血肉相连，在祖国西南边疆筑起一道钢铁长城。

（三）

当前，西藏无产阶级文化大革命和全国一样，形势大好，而且越来越好。活学活用毛泽东思想的群众运动空前大发展，光芒四射的毛泽东思想空前大普及，毛主席一系列最新指示更加深入人心。广大革命群众，积极批判反动的资产阶级的“多中心论”，在以毛主席为首、林副主席为副的无产阶级司令部的英明领导下，统一认识，统一步伐，统一行动。全区已办了10200多期各种类型的毛泽东思想学习班，有40万4千余人次参加了学习，“斗私，批修”，促进了人的思想革命化。革命大批判、革命大联合和革命“三结合”正迅速发展，西藏两大革命群众组织已于3月14日达成大联合协议，8月21日又达成归口大联合协议。全区开展“四大”的49个县有39个县已实现了革命大联合，部分县成立了革命委员会。全区军民意气风发，斗志昂扬。“抓革命，促生产，促工作，促战备。”在去年农牧业丰收的基础上，正在为夺取第十个丰收年而努力。

综观全区形势，成立西藏自治区革命委员会的条件已经成熟。早日成立革命委员会，是西藏130万各族革命人民和驻藏人民解放军广大指战员最迫切的愿望。

有关西藏自治区革命委员会的组成，经我们同两大革命群众组织充分协商，提出如下建议：

一、革命委员会由135人组成。名额分配是：

1.革命群众代表68名（其中工人代表22名，贫苦农（牧）民代表10名，红卫兵代表10名，其他方面的代表26名），占50.4％；68名革命群众代表中藏族29名，女同志11名。

2.军队代表20名，占14.8％。

3.革命领导干部20名，占14.8％。

4.机动27名（主要增补群众代表和革命领导干部），占20％。

二、革命委员会的常委由35人组成。其中革命群众组织代表19名（暂缺1人），军队代表8名，革命领导干部8名（暂缺4人）。

三、建议由曾雍雅同志任主任。由任荣、陈明义、廖步云、杨东生、苗丕一、阿沛·阿旺晋美、巴桑、陶长松、刘绍民、土登尼玛、朱景尚、马瑞华、次仁拉姆等13人任副主任，另留副主任2名，以后增补。

四、根据伟大领袖毛主席“精兵简政”的教导，自治区革命委员会工作人员拟定为100人左右。设办事组、政治工作组、生产指挥组、人民保卫组。

以上报告，请审查批示。

中共西藏军区委员会1968年8月28日
