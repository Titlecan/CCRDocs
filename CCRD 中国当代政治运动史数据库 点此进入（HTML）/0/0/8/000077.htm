中央文革关于进一步批判大毒草《刘少奇访问印度尼西亚》的通知

日期：1968-11-18 作者：[待确定]

各省、市、自治区革命委员会：

当前全国正处在揭发、控诉、批判刘贼叛党卖国滔天罪行的高潮中。为了配合大批判，各地可有领导地重新放映刘贼《访问印度尼西亚》（不登广告），并组织群众性的大批判，报上也可发表批判文章。我们即将再印制一批拷贝送发各地。

中央文革一九六八年十一月十八日

来源：

根据湖北省革命委员会编印《无产阶级文化大革命文件汇编[三]》，武汉：1969年3月
