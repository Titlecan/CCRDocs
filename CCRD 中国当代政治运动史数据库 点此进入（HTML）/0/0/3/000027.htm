中共中央、国务院批转中央安置工作领导小组关于城市精简职工和青年学生安置工作领导小组长会议的报告

中共中央、国务院批示：现将中央安置工作领导小组关于城市精简职工和青年学生安置工作领导小组长会议的报告，发给你们。中央、国务院基本同意这个报告，请研究执行。

分散插入人民公社生产队，是今后安置城市下乡生产人员的主要方式，各省、市、自治区要认真总结这方面的经验，以利今后工作的开展。

今年各大中城市暑期毕业不能升学、不能在城市就业的高初中学生，估算约三十余万人，安置任务很大，要求各省、市、自治区早作研究，早作安排。

要求各省、市、自治区拟定一九六四年的安置计划和十五年规划轮廓，准备在九月召开的城市工作会议上讨论。

关于城市精简职工和青年学生安置工作领导小组长会议的报告

(一九六三年七月三十日)

谭副总理并报总理、国务院、中央：

遵照谭副总理指示，于六月二十九日到七月十日，召开了六个大区城市精简职工和青年学生安置工作领导小组长会议，参加这次会议的还有十一个重点省、市主管安置工作的同志。会上检查了一年来国营农、牧、林、渔场安置城市精简职工和学生的工作，总结了这方面的经验。在会议期间，总理和谭副总理接见了各大区的到会同志，听取了汇报，对今后工作作了重要指示。总理指出，今后十五年内动员城市青年学生下乡参加农业生产，是城乡结合、移风易俗的一件大事。要求各大区、各省、市、自治区都要作长远打算，编制十五年安置规划。今后安置的主要方向是插入人民公社生产队，其次，是插入国营农、牧、林、渔场，再其次，才是建立新的国营农、牧、林、渔场；并且要各级安置工作领导小组把插队工作统管起来。到会同志一致认为：这次会议收获很大，进一步提高了对安置工作的认识，明确了方向，并深深体会到全面总结经验的重要意义。今后要认真地、踏实地、不断地总结经验，提高认识，使我们的工作不断前进。

(一)安置工作的情况和体会

近两年来，从城市精简回农村的约二千六百万人(劳动部的统计)。其中，绝大多数是家居农村，回乡生产的；一部分是久居城市，安置到农村人民公社生产队或国营农、牧、林、渔场，参加农业生产的。安置到农村分散插队的人数有多少，还没有全面地统计。据浙江嘉兴县的材料，一九六二年下乡插到生产队的有五千三百多人，分在二千多个生产队里，百分之九十五以上巩固下来了。经验证明，分散插队参加农业生产，是安置久居城市的精简职工和青年学生的一种主要方式，花钱少，收效大，多快好省。一年以来，安置到国营农、牧、林、渔场的，到今年五月底止，已达十五万四千人，占原计划三十三万二千人的百分之四十六强，巩固的程度更高一些，返回城市的仅占百分之二。这也为安置家居大中城市的精简职工和青年学生的工作积累了一些经验。

这次会议，对国营农场(包括林、牧、渔场，下同)一年来的安置工作进行了检查，也研究了嘉兴县的下乡插队的经验。基本经验有两条：一是加强具体领导，一是把思想工作做深做透。这两条贯串到安置工作的各个环节。必须做好思想动员，又要解决具体问题，坚持自愿原则，依靠群众做好安置工作，准备好再安置，做到个人自愿，群众欢迎，再加上政府必要的和可能的扶助，达到安置好，巩固好的目的。

一、首先做好思想动员工作。根据去年中央“关于妥善安置和热情接待下乡职工的通知”和“关于减少职工和城镇人口的宣传要点”，广泛深入地宣传，使党的以农业为基础的方针深入人心，改变轻视农业劳动的旧习惯，树立参加农业劳动的新风尚。使人们了解，城市人员特别是青年学生下乡落户，不仅为农业生产的发展提供了劳动力，而且是城乡结合，知识分子与劳动人民结合，移风易俗的一件大事。农村是青年锻炼的场所，大有前途。并且，在一般动员的基础上，进一步下苦工夫，深入思想动员，进行个别串连，发现和培养积极分子，解除他们的思想顾虑，解决他们的实际困难，做到四通：本人通，家长家属通，亲邻朋友通，老师同学通，让他们心情舒畅地下乡落户。苏州市动员六百多学生到苏北农场去，就是这样做的，效果很好。另一方面，也要做到社队和农场干部通，也要向社队和农场干部，向社员和农场职工，普遍深入地进行宣传教育工作，讲大好形势，摆暂时困难，交代政策，使他们自觉地承担安置任务，依靠他们，做好安置工作。浙江嘉兴县在这一方面做得很出色，解除了某些人思想上的疑虑，表明久居城市的人也可以下乡插队，参加农业生产；他们到农村落户，也能够巩固下来；农村既能安置强劳动力，也能带家属和辅助劳动力，一起到农村落户。

二、组织好，准备好，安置计划要落实到两头。一头落实到城市，落实到准备安置下乡的每一个人，根据城市需要安置到农村去的人数，以及这些人的思想情况和动员工作的情况，提出分期分批下乡的具体安排。一头落实到生产队或农场，摸清由哪个公社生产队安置，由哪个农场安置，能够安置多少户，多少人，接待的准备工作做得怎样，什么时候去比较适宜。然后把这两头的安排对对口径，使城市人员下乡的安置同生产队、农场的接待准备工作衔接起来，实行城乡挂钩，地区挂钩，双方联系好，有组织地把需要安置的人员送到生产队和农场落户。上海市对于去安徽插队的人员，采取了先逐户落实到队，而后有组织地迁移的办法，准备充分，安置妥善，结果双方满意。

安置到国营农场，是成批下去，集中安置的，同分散插队不同，一定要按照规定事先准备好居住的房屋，对于他们的工资待遇、口粮标准都要有具体的规定，他们的生活用品的供应也要落实。还要由城市配备好适当的带队干部，一起和他们下农场落户，至少是在农场工作一个时期，帮助农场做好对这些来场落户人员的思想政治工作。

安置到公社生产队和国营农场的人员，绝大部分是参加农业生产的，也有少数是根据队和场的需要和下乡人员的具体条件，分别担任社队的会计、抽水机等农业机械的司机手、社队办的小型企业的职工、农场的基建队伍和加工厂的职工，还有安置到营林新村落户的。这些门路，既多安置了一些人，又利用了各人的特长，使他们各得其所。

三、做好巩固工作，使下乡人员在农村、农场安居乐业，是安置工作的最终目的，也是接收单位的一项长期艰巨的工作。根据各地经验，无论是分散插队或者集中安置到农场的，都必须加强政治思想教育，不断提高他们的政治觉悟和劳动积极性。以党团支部为核心，依靠贫下中农和农场老工人，开展以老带新、新老互助的活动，帮助他们过好五关：思想关、政治关、家庭关、社会关、生活(劳动)关。组织贫下中农向他们讲村史，讲旧社会的苦，组织农场老工人向他们讲勤俭办场的历史，通过活的阶级斗争和生产斗争来教育他们，帮助他们从思想上、政治上在农村安家落户，决心红在农村，专在农村。

依靠群众，帮助下乡人员安家，特别要注意帮助单身青年学生解决生产生活中的困难问题。对于分散插队的，要依靠老社员，分别采取挤、借、租、让、建等办法，解决他们的住房问题；帮助他们解决自留地、口粮和农具等问题，还要教会他们农业生产技术；对于单身的青年，还要采用搭伙等办法，解决他们的吃饭问题。嘉兴县就是这样做的，县委还定期检查，发现问题，及时解决，使下乡人员的生活基本上得到保证。

为了教会下乡青年学生的农业生产技术，农场的做法是，先集训两三个月，半耕半读，边学习边劳动，劳动也是由轻到重，由简到繁，使他们逐步习惯于农业劳动。在他们正常参加农业劳动以后，还要注意城市学生“好学”的特点，有计划地组织他们进行业余学习，使他们的文化水平逐步提高。苦干三年，头年打底，二年自给，三年自立，是每一个下乡落户、参加农业生产的人员所应具有的决心。

离开城市到农村参加农业生产，是会遇到一些困难的，特别是青年学生，难免中途思家，又想返回城市。社队和农场干部，社员和农场工人，应该发扬互助友爱的精神，积极主动地帮助他们，稳定他们的情绪，而不可讽刺打击他们。各级党政机关和群众团体，要经常进行检查访问，帮助解决他们的实际问题；还可以在适当时间，采取下乡人员代表会，座谈会等形式，表扬好人好事，鼓励他们；或者利用下乡学生回家探亲的机会，召开群众大会，一方面由下乡学生报告劳动生活情况，同时请到会家长代表讲话，鼓励他们，安定他们。经验证明，这些方式，对于下乡落户人员的巩固工作，作用很大。

近两年来，安置工作的经验是丰富的，但是，这次会议时间仓促，原来只准备讨论国营农场的安置工作，对于分散插队的经验准备的材料不多，研究不够，有待于另开一次会议进一步加以总结。

(二)今后安置工作的安排

一、遵照总理和谭副总理指示，今年下半年原定安置十八万人的计划，要力争超过，不得减少。在保证实现原定计划的前提下，安置系统(农、牧、林、渔等等)、安置方式(插队还是下农场)和安置对象(精简职工还是青年学生等等)，需要调整的，可以由省、市、自治区决定，作适当调整，报中央和中央局的安置工作领导小组备查。有些省、市要求增加任务，可以另作计划，由中央局审核，报国务院批准。

今后需要安置的，主要是不能升学和就业的初高中毕业生，精简职工的人数估计不会很多。安置下乡的学生，年龄可以由十八岁放宽到十六岁，条件是能够独立生活。主要是安置大中城市的学生，县城里不能升学和就业的学生也可以适当安置一些。由县城安置下乡的，是在本县农村就近安置的，安置费的开支应该低一些。达到劳动年龄的不能升学的农场职工的子女，应该由农场自己负责安置。对于城市闲散人员，安置下乡生产的条件不要规定过死，家庭生活比较困难、急需就业又自愿下乡、下场的，都可以安置。凡是参加生产队或者农场劳动的，都应该给予合理的报酬。安置到人民公社插队的，安置费的定额应该低于农场安置费的定额，可以沿用精简小组的规定，以每人一百元为标准，由省、市、自治区根据县、社、队不同的实际情况，提出所需的经费，报中央局核定。采取分散插队的办法，按原定的安置费，可以多安置一些人。

二、我国城市人口自然增长率很高。从现在起就大力推行计划生育，只能逐步降低今后的人口增长率。解放十四年来出生的孩子，在今后的十五年内，将分批逐年达到劳动年龄，单就二百个左右大中城市计算，每年大约有二百多万人。除城市就业、参军和自动下乡的一部分以外，估计每年有百万左右的人需要有计划地安置下乡，参加农、林、牧、渔、副业生产。他们一般是高中、初中毕业，不能升学的学生。每年暑期，城市就会感到这方面的压力很大。分期分批地有计划地把这些青年学生安排到农业生产战线上去，是摆在我们面前的一项长期的艰巨的任务。因此，要求各省、市、自治区，以安置城市里不能升学的、达到劳动年龄的青年学生为中心，制定出十五年的安置工作的规划。

三、今后安置下乡人员的主要方向，是插到人民公社生产队去。全国有五百四十多万个生产队，在十五年内，每队先后插入两个人，就可以安置一千多万人，根据嘉兴县的经验，这是完全可以办到的，也是多、快、好、省的办法。其次，是安置到国营农场和场社合并的农场。再其次，是扩建、新建一些农场。有的山区，过去被国民党破坏，人口锐减，至今仍未恢复，例如福建、江西、湖南还有这样的地方，全国其他的老根据地都有这样的地方。在这种地方可以仿照吉林办营林村的经验，采取“国家扶助、集体所有、集体分配”的办法，安置一部分人，或按照江西举办小型国营垦殖场来进行安置。这样，既安置了城市下乡人员，又发展了山区经济，恢复了老区。

四、要求各省、市、自治区做出十五年的安置规划，经大区审核后，提交九月城市工作会议上讨论。一九六四年的安置人数、所需物资和投资，应提早做出具体计划，于九月以前报来，以便纳入明年国家的计划。

对于编制十五年长远规划，提出如下参考意见：第一，要摸清各大中城市各级(由高小到高中)、各类(普通中学和专业学校)的逐年毕业人数，对照招生计划做出不能升学人数的概算，加上闲散人员和精简的职工，减去城市可能就业的人数，计算出需要安置下乡的总人数和分年的人数。然后根据安置的需要安排接收单位，定出分年安置的计划，以便今后逐年提早协商，真正做到两头落实。可以先搞出明年的计划，明年的计划要比较具体，然后在明年计划的基础上，再设想拟制五年的十五年的计划。第二，在编制计划时，以省、市、自治区为单位，按照省内就地安置的原则，把本省各大中城市需要安置下乡的人员安排好。县城接近农村，需要安置的人员下乡插队也比较容易，可以由县自理。京、沪两市分别由华北局、华东局统筹，跨省安置；还可以考虑由新疆生产兵团和东北农垦总局分别适当安置一批。第三，在有条件的省区，从长远打算，可以考虑同开垦荒地、扩大耕地面积、发展农业生产结合起来，有重点地垦荒，安置城市下乡人员，例如开发广东、河北的沿海滩地，安置广州、天津两市的下乡人员。

这项安置计划，经审定后，应该纳入国家的年度计划和长远计划，要求各有关方面共同努力完成。

五、安置城市下乡人员的工作，由各级农办负责领导。各级安置工作领导小组，可以在原有基础上加以扩大，吸收有关部门和群众团体参加，加强领导。并且要在各级领导小组下建立办公室，作为常设的办事机构。中央安置工作办公室由农垦部负责兼办。

安置城市下乡生产的人员，主要是安置不能升学、不能在城市就业的青年学生，下乡参加农业生产，是一项长期艰巨的工作，而且是一项新的、经验还很不足的工作，一定要加强领导，配备起一个强有力的班子，把这项工作做好。安置工作也要做到五好：教育好、组织好、准备好、安置好、巩固好。使下乡落户、参加农业生产的青年学生，在贫下中农和农场工人的教育影响下，真正成为有社会主义觉悟、有文化、有技术的新型的农业劳动者。
