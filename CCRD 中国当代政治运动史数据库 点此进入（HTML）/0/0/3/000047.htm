中央宣传部关于转发总政宣传部“关于宣传雷锋事迹问题的报告”的通知

中央宣传部批语：

我们同意总政宣传部关于宣传雷锋事迹问题的报告。现在转发给你们，请参照执行。

关于宣传雷锋事迹问题的报告

中央宣传部：

宣传和学习雷锋事迹的活动，目前已经在全军、全国青少年和机关干部中大规模地展开，许多地方形成了热潮，收到了极好的效果。

最近，根据湖南省委宣传部在雷锋家乡进行的调查，发现关于雷锋童年时期的报导，在个别情节上与事实有出入。为了保持和发扬我党实事求是地进行宣传的优良传统，和避免因为个别情节的失实而损害雷锋的英雄形象，我们对今后进一步的宣传活动提出以下意见：

一、目前，我们正会同有关部门，根据各方面的反映，对解放军文艺社正在编印的《雷锋日记》、《雷锋的故事》和中国青年出版社正在编印的《毛主席的好战士——雷锋》进行审查和订正。这三本小册子不日即可出版发行。各地正在编印的有关雷锋事迹的各种出版物和宣传品（包括广播录音、幻灯、图片、连环画以及用雷锋命名的文艺作品等等），凡是没有出版发行的，都应该根据上述小册子进行订正。订正的时候不必作什么声明，以免引起读者不必要的猜疑。已经出版发行的，则不必更正。

二、关于雷锋事迹的宣传，今后应该讲求质量，不要追逐数量。在宣传内容上，应该着重宣传雷锋同志参加革命以后的模范事迹，以及他的共产主义思想品质。对他的童年时期和他的一些亲属不要过多地去作文章。更不要抢先发表未经核实的新材料。

以上意见，如果认为可行，请转发各省市和有关单位参考。

总政治部宣传部1963年3月9日

来源：中共中央宣传部办公厅编：《党的宣传工作文件选编》，中共中央党校出版社，1994。
