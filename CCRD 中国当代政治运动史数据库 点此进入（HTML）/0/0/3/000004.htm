中共中央、国务院关于阜新煤矿物资器材被盗问题的指示

各中央局，国务院各部、委，各省、市、自治区党委和人委：

现将中共辽宁省委《关于贯彻执行中央负责同志对阜新煤矿物资器材被盗问题的批示情况报告》，转发给你们。

从辽宁省委的报告中可以看出，目前在企业、事业单位中国家财产被贪污盗窃的情况，是相当严重的。在有些单位，这种破坏社会主义建设的活动，十分嚣张，十分严重。阜新煤矿就是一个典型例子。这个煤矿，目前已经查出：一九六一年以来发生的各种盗窃、破坏案件和线索，就有三百五十六起，其中重大的有九十一起。盗窃分子不择手段，或者监守自盗，或者内外勾结，甚至拆毁设备、破坏生产。

盗窃厂矿物资器材的犯罪活动，直接地严重地损害着社会主义全民所有制的经济，并且腐蚀着一部分干部和工人，这本质上是资本主义对社会主义的猖狂进攻，是资产阶级向工人阶级的猖狂进攻，是激烈的阶级斗争的一种反映。如果不彻底粉碎资产阶级的这种进攻，任其发展下去，社会主义全民所有制的经济就不能得到可靠的保障，而且有使资本主义势力在某些单位复辟的危险。应当教育企业事业单位的广大干部和工人，对这个问题引起严重注意，提高阶级觉悟和警惕性。

从阜新市反贪污盗窃斗争的经验来看，情况确是十分严重的。在某些企业中，拿、摸公物的人数一般占职工总数百分之三十至五十，甚至有的占百分之七八十或百分之百，并且其中不少与其他企业和社会上惯盗惯窃和五类分子有所勾结。但是，如果加以分析、加以区别，那么，情况就有所不同。根据典型调查，除去一般占公家便宜这种不良作风以外，属于偷、摸性质的约占职工的百分之四，真正的贪污盗窃分子则不到百分之一。问题是为什么在党领导的社会主义企业中，会发生这样普遍而且长期存在的不良和违法现象呢？中央和国务院认为，这是由于某些企业和地方领导干部滋长了严重的官僚主义作风。在最近几年来生活困难的掩盖下，忽视了政治挂帅和政治思想教育工作，忽视了在企业内部和社会上还有资产阶级思想和旧社会遗留下来习惯势力的不小影响，还存在着两条道路的阶级斗争。对企业内部和社会上长期存在的贪污盗窃活动和不良作风熟视无睹，放任自流。甚至有少数领导干部也被资产阶级思想侵蚀，丧失了立场，腐化堕落，蜕化变质，参与贪污盗窃活动。因此，反对贪污盗窃和肃清不良作风的运动，必须政治挂帅，领导带头，放于发动群众，实行领导、干部、群众三结合，划清一般占公家便宜和偷、摸公家财物的界限，划清一般偷、摸和贪污盗窃的界限，结合增产节约运动，团结和教育多数职工，孤立和打击少数贪污盗窃分子，为下一步企业开展“五反”运动创造有利条件。一些有问题的企业领导干部，必须认真带头检查交代，决不能姑息迁就。辽宁省委报告中没有着重提到整顿领导作风、领导干部带头检查的问题。在这里，应当特别指出，关键在于领导，必须认真整顿领导作风，改进企业领导工作，这是进行反对贪污盗窃、肃清不良作风和今后开展增产节约和“五反”运动的重要保证。

希望各地方、各部门根据中央关于厉行增产节约和反对贪污盗窃、反对投机倒把、反对铺张浪费、反对分散主义、反对官僚主义运动的指示，检查企业、事业单位和机关团体中的偷窃破坏现象和占公家便宜的不良作风，并且积极准备在运动展开以后在所有单位里，充分发动群众，彻底揭露和坚决打击一切贪污盗窃分子，坚决反对一切损害社会主义事业的行为。辽宁省委报告中的经验和他们对几个有关政策界限和工作方法问题的意见，可以供你们参考。

中共中央国 务 院一九六三年四月三日（发至县级）
