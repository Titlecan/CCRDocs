中共中央关于在未土改地区进行民主革命补课问题给青海省委的批复

日期：1963-5-26 作者：[待确定]

青海省委并西北局：

五月十七日报告悉。同意你们在未土改地区进行民主革命补课的意见。在进行这个工作时，应当和当前农村社会主义教育运动结合起来。在政策上、步骤上和方法上都必须经过典型试验、取得经验后，才分期分批进行。在方法上要认真进行扎根串连，把根子扎正扎稳；在时间上不能要求过急，可以在年内做完，也可延长到明年。总之，民主革命补课工作必须做，必须做好，必须根据最近中共中央《关于目前农村工作若干问题的决定（草案）》的精神进行。

中 央一九六三年五月二十六日
