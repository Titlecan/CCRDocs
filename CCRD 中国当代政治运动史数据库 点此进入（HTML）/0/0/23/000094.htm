广西壮族自治区党委组织部关手进一步做好调整配备领导班子工作的通知

日期：1983-11-28 作者：[待确定]

各地、市委组织部，区直各单位党组（党委），区党委工作组：

当前，我区各地、市和区直局以上单位，正在深入进行机构改革、调整配备地、市和区直部、委、厅、局一级领导班子的工作，有的单位已经开始进行二层机构和职能处室领导干部的调整，总的进展情况是好的。根据区党委批准的《在处理“文革”遗留问题中抓紧机构改革调整好区直部委办厅局和地市领导班子的方案》（讨论稿），以及中组部组通字（83）50号文件的要求，为了把好政治、文化、年龄三关，充分走群众路线，认真尊重民意，防止派性干扰，进一步做好调整配备领导班子工作，现重申以下几点：

一、驻各地、市和区直各单位的区党委工作组，在党委（党组）领导下，共同商量好具体组织民主推荐和民意测验的办法，广泛听取群众的意见，然后，集中民主推荐和民意测验情况向党委汇报。

二、要征求党性强，没有派性，办事公道，群众信得过的老干部对调整配备领导班子的意见。具体作法可采取个别了解或召开座谈会的形式进行，采纳他们正确的意见，对没有采纳的意见应耐心解释，说明道理，不要挫伤老同志的积极性。

三、在区党委审批前，先上报区党委审批任职的干部名单，同时报送干部任职呈报表、群众推荐对象名单及民意测验统计数字。对呈报任职对象的每一个阶段（①“四清”中及“文革”开始时；②批“资产阶级反动路线”时；③夺权时；④武斗时，⑤“清理阶级队伍”时；⑥清查“5·16”时多。⑦“批林批孔”时；⑧“反击右倾翻案风”、“天安门事件”时；⑨十一届三中全会以来。）的详细考核材料，一式二十五份送区党委组织部，以便上报区党委审议。许多单位对此项工作是注意的，也做得比较好，但也有个别单位由于组织部门抓得不够紧，或是缺乏经验，漏洞不少，造成审批工作失误和不能及时进行，望引起注意。

中共广西壮族自治区委员会组织部一九八三年十一月二十八日

来源：

( 《处理“文革”遗留问题、清理“三种人”文件汇编》，广西壮族自治区党委整党办公室，一九八六年八月。)
