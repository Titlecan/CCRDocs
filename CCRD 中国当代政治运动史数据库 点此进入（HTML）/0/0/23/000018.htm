日期：1983-5-13 作者：[待确定]

自治区直属机关各都、委、各办、厅、局及各人民团体党组，各大专院校党委：

自治区党委运动办《关于下达自治区直属机关行政事业单位落实政策经费分配方案的报告》，已经区党委领导原则同意，现转发给你们，请参照执行。

内蒙古党委办公厅1983年5月13日

关于下达自治区直属机关行政事业单位落实政策经费分配方案的报告

自治区党委：

在十年动乱期间，我区广大干部、群众，特别是少数民族干部、群众，受到严重迫害。党的十一届三中全会以来，区党委拨乱反正，平反了大量冤假错案，使广大干部、群众在政治上重新翻身。为了认真落实党的政策，促进安定团结，1978年底，党中央拨给我区行政事业单位在“文化大革命”中受害干部职工落实政策专项经费1600万元。1980年11月7日，经自治区政府主席办公会议研究决定：将其中的1130万元分配给各盟市使用，此款已于同年底下达；分配给自治区直属机关的70万元，还没有分配下去。

两年来。自治区直属74个行政事业单位中有59个单位自筹资金共34万3400元，用于补助受害者1067人，使大部分受害者感受到了党和国家的关怀，同时也程度不同地解决了他们的生活困难，调动了他们工作、生产的积极性。由于各级党委、党组的重视，经费使用的情况总的是好的，但也存在一些问题。主要是单位与单位之间很不平衡，受害者所得补助也苦乐不均。如自筹万元以上的10个单位中，民委自筹12784元，补助给受害者29人，人均440元；出版局自筹17318元，补助52人，人均333元。有的个别受害者得补助高达4500元。有的只是受到一点触及，也得了补助。而没有自筹能力的（大部是行政单位）15个单位的受害者，则分文未补。特别值得注意的是，有的造反整人的，也混水摸鱼得了补助。这虽属个别现象，但在群众中造成了极坏的影响。

产生上述问题的原因是多方面的。主要是工作上的问题。有些单位对落实政策补助经费使用的指导思想不很明确，片面强调经济补助，忽视了思想政治工作；有的工作做得不细，缺乏认真的调查研究，对多要的多给，少要的少给，没有申请的不给或少给；有的把关不严，使一些不该补的人钻了空子，得了补助。

最近，直属机关74个行政事业单位申报要求补助的达2489人，补助费数额达121万6900元。个人申请补助少则4元、6元（财经学院），多的要求补助15000元和17000元。从各单位申报的情况中，我们还发现，由于过去政策掌握的很不一致，有不少人单位已给过适当补助，这次又申请补助。

为了使用好直属机关70万元落实政策经费，切实解决受害较重的干部、群众的一些生活困难，我们在反复调查研究，认真总结过去经验的基础上，提出如下分配方案：

一、补助经费的分配原则

落实政策专项经费，只限于解决自治区直属机关行政事业单位的干部职工在“文化大革命”中因冤假错案受害严重而造成的生活困难。对受害重、损失大、困难多的给予一次性补助。要实事求是，区别对待，不搞平均分摊。该多补的多补，该少补的少补，不该补的绝不补。凡过去已经补助过的这次原则上不再补助。生活确有困难的可以酌情适当补助。厅、局长以上领导干部原则上一般不予补助。

二、补助的重点

1.运动中受害致死，包括严重伤残因旧伤复发后定为因公死亡的。

2.严重伤残完全或基本丧失工作能力和劳动能力的。

3.查抄经济损失严重的（有人证明或有其它证据的）。

4.错关错扑，包括军管关押，身心受到严重摧残的。

5.其他受害严重、经济损失大的。

三、几点要求

1.中央拨给我区的落实政策专项经费，是党和国家对受害干部、职工的关怀。各级党委、党组，应当明确认识，落实政策首先是政治上的平反，经济上只是作适当的补助。经济补助不是搞救济，也不完全是搞经济损失赔偿。因此，要深入细致地做好受害同志的思想政治工作，教育他们互谅互让，顾全大局，团结一致向前看。受害较重的领导干部，要带头体谅国家困难，用实际行动，带动其他受害的同志，发扬革命风格，把有限的补助经费用在受害重、生活困难大的同志身上。

2.落实政策经费以厅局为单位下达，区党委运动办一律不对受害者个人。对受害者的补助，原则上由原单位负责；原单位已撤销的，由原单位上一级机关或现单位负责。不论受害者现在是否在原单位工作，评议时都要一视同仁，同等对待。要防止不负责任和互相推诿的现象发生。

对党外知名人士和高级知识分子中的受害者，在同等条件下，可优先补助。

对个别人想借机捞一把，从原单位和现单位领取双份补助费，发不道德之财、损人利己之财的，要严肃对待，对其进行批评教育。要坚决杜绝造反整人的人得到补助；过去已补助的，一经发现，要坚决追回，并视其情节轻重，经予必要的处理。

3.落实政策补助经费分配涉及面广，政策性强，关系到受害干部职工的切身利益，必须认真对待，专款专用。各单位可在调查摸底的基础上，采取自报公议，党委（党组）严格把关，集体审定，最后张榜公布的办法。发现问题，要及时纠正。对因工作不负责任，违背经费分配原则，徇私舞弊，或不经集体研究，私自批条子，在群众中造成极坏影响的，要追究有关人员及领导的责任。

这次落实政策补助经费，采取包干分配的办法，一次分拨给各单位。各单位要加强领导，负责到底，把有限的补助经费妥善安排好，认真做好各方面的工作，发现问题，及时解决，切实把这项工作抓紧抓好。

以上报告如无不妥，请批转有关单位执行。

内蒙古党委运动办公室

附：落实政策经费分配方案表

1983年5月25日
