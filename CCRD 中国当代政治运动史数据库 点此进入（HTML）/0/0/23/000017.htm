中共中央对《关于广西召开区党委工作会议的情况报告》的批复

日期：1983-4-30 作者：[待确定]

广西自治区党委：

你们四月二十日《关于广西召开区党委工作会议的情况报告》收悉。中央认为，这个《报告》很好。对广西未能及时处理好“文化大革命”遗留问题症结的分析和认识是正确的。乔晓光同志在总结中的检查是好的。

中央相信，你们只要根据区党委工作会议的精神积极工作，具体落实会议提出的各项措施，广西的遗留问题一定能够得到妥善解决，开创广西工作的新局面大有希望。

中共中央一九八三年四月三十日

来源： 《处理“文革”遗留问题、清理“三种人”文件汇编》，广西壮族自治区党委整党办公室，一九八六年八月。

来源：

( 《处理“文革”遗留问题、清理“三种人”文件汇编》，广西壮族自治区党委整党办公室，一九八六年八月。)
