广西壮族自治区党委处遗领导小组办公室关于区高等院校在“文革”中死亡的毕业生的善后经济处理问题给区教卫办处遗领导小组的复文

日期：1983-12-14 作者：[待确定]

区教卫办处遗领导小组：

你办桂教卫遗字[1983]11号关于我区高等院校在“文革”中死亡的毕业生的善后经济的处理意见，经请示区党委，现答复如下：凡属“文革”中已毕业应分配而未分配，因参加“文革”被打死的大专毕业生，其善后待遇可按干部处理。请你们转告所属院校按区党委桂发[83]55号文件的有关精神执行。

来源：

( 《处理“文革”遗留问题、清理“三种人”文件汇编》，广西壮族自治区党委整党办公室，一九八六年八月。)
