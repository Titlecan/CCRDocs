国务院知青办、财政部关于转发安徽省财政局、知青办《关于检查知青经费使用情况和意见的报告》的通知

日期：1979-5-28 作者：[待确定]

各省、市、自治区知青办、财政局：

现将安徽省财政局、知青办《关于检查知青经费使用情况和意见的报告》转发给你们，供参考。

去年以来，各地相继开展了知青经费、物资大检查，取得了较好效果，要继续把这项工作抓好。对前一段查出来的问题要严肃处理；检查得不深不透的要搞好补课；还没有行动起来的要组织力量，集中时间，抓紧进行。

通过这次大检查，各地要认真总结经验教训，严格财经纪律。在知青经费、物资的管理使用上，要认真执行中央〔1978〕74号文件的有关规定和最近国务院知青办、财政部《关于知青经费管理使用的暂行规定》。严禁挥霍浪费、贪污盗窃、侵占挪用。

安徽省财政局、知青办关于检查知青经费使用情况和意见的报告(摘要)

(一九七九年三月二日)

最近我们在蚌埠市召集地、市知青办主任、会计和财政局分管这项工作的同志开了个汇报会，重点了解各地对县以上知青单位进行知青经费、物资检查的情况。从各地检查的结果看，知青经费、物资的管理使用存在不少问题，主要是：

1．虚报、重报下乡知青人数，冒领经费。据不完全统计，全省有十五个市县多报知青人数二千九百零三人，冒领经费一百四十余万元。桐城县多报六百二十人，冒领经费二十九万七千元。

2．挪用知青经费盖办公大楼和接待站。据不完全统计，共挪用一百二十九万余元。如阜阳地区和所属六个县盖了办公大楼和接待站，动用知青经费二十九万二千元，木材二百零九立方米。淮北市杜集、烈山、相山三个公社盖大楼和搞社办企业，挪用知青经费二十五万五千元。

3．任意开支购买高档商品。据不完全统计，动用知青经费四十二万一千元。全省大部分地、市知青办动用知青经费购买了小汽车，有的还购买了电风扇、电视机、收音机等高档商品。淮北市于一九七五、一九七七两年购买吉普、旅行车各一部，用去知青经费六万余元。太和县知青办现有十三人，购买自行车十八辆。

4．吃喝招待，铺张浪费。据不完全统计，各地用于招待费共花掉九万四千一百元。还有送礼的，如芜湖市送给社队干部黄背包二百三十个，花掉五百零四元；金寨县知青办派人护送县革委负责人去沪看病，用知青经费购买土特产品送人。

5．贪污知青经费。池州地区发现贪污知青经费三十九起，计四千六百多元。六安地区据不完全统计，区、社干部贪污知青经费二万多元，六安县发现贪污五百元以上的就有七人，其中新安区知青专干以假单据报销贪污知青款一千七百余元。苏家埠区会计贪污知青款三千零一十元，蚌埠市知青办会计、和县知青办办公室主任贪污都在千元以上，目前正在进一步查处。

6．职工借支知青经费长期不还的数字很大，据不完全统计有四万二千七百元。

7．知青住房及三具损坏、丢失和占用现象也比较普遍。据池州地区统计，损失价值二十四万七千二百元，其中属于知青丢失、带走的五万四千九百元，私人占用的四万三千七百元，集体占用的七万五千元，损坏的七万三千六百元。

以上问题，大部分是从县以上知青单位检查出来的，公社以下的问题还要多一些，还需进行一次认真清理。

根据省委〔1978〕73号文件精神，各地对前一阶段已查出的问题应认真进行处理，少数县级单位检查得不深不透的要进行补课。下一阶段，主要是检查和清理公社及公社以下单位的知青经费使用情况。

从各地已清理的情况看，有大量空闲房屋、三具和一部分结存经费需要清查处理。据统计全省为知青建房共达二十三万间，其中砖瓦结构的十二万多间，一九七三年至一九七八年用去建房费七千六百六十余万元，三具费二千零三十万元。匡算全省公社一级结存经费达一千万元。今后城市知青改为广开门路安置，集中下放到知青场队，过去在农村盖的房屋将大部分空闲，如不及时进行清理和处理，势必使国家财产造成很大损失。

公社以下知青经费清理方法，以公社组织力量自查为主，县组织互查或重点检查，清查的年限主要是从一九七三年起，有些一九七三年前下乡知青多的公社也要一并清查。由于公社以下知青经费的使用牵连面广，管理混乱，情况复杂，问题较多，可边检查边清理边处理，不然容易形成走过场。在清理过程中要健全手续制度，收回房屋、“三具”变价款等要严格履行财务手续，逐笔登记，由公社开三联单立据。

对检查出的问题，提出如下处理意见：

1．以虚报、重报知青人数冒领的知青经费和物资，要抵作新下乡知青使用，没有知青下乡任务的，经费、物资要上缴。挪用知青经费建办公大楼、招待所的，应交给县革委会统一安排使用，动用的知青经费要归还。

2．县办工厂及社队办企业动用知青经费的，要订出还款计划，限期偿还。

3．对集体或个人挪用、借用的知青经费物资应全部归还，特别是知青工作干部的欠款应以身作则，积极带头归还。对贪污盗窃要严肃处理，追回赃款。对动用知青经费购买的高档商品和多余物资应作价处理，价款仍作为知青经费。

4．认真处理好空闲的房屋。除了少数调给已婚知青居住外，一律作价处理，原则上先公后私，主要卖给社队，在特殊情况下，也可以卖给私人。房价按国家、社队投资(包括用工)总数和房屋新旧程度折价计算，并按照国家和社队投资的比例，计算出国家和社队各得价款，折价要合理，不得变相私分。对知青空闲房屋，须由公社统一报县知青领导小组批准后才能处理。

知青因“三招”离队移交的炊具、用具和农具，也要全面进行清理，闲置的“三具”可以按质论价卖给社队和个人。

5．各县清理收回的知青经费，省、市两级不要，地区可以少留或不留，大部分归县，少部分给公社。留给公社的比例可以不同，有的公社清理收回的经费数字大，而在乡知青人数又少，留成比例应该小一点，具体比例由各地自行确定。清理收回的经费除用于补助在乡知青生产、生活困难和已婚知青的困难外，其余的用作扶持知青场队的生产资金，不准挪作他用。

各级革委会要重视知青经费的检查和处理工作，要有一位负责同志亲自抓，县知青办和财政部门要密切配合，组织力量(可以抽一些在乡知青帮助清理，工资在业务费开支)，集中一段时间，狠抓落实。要求一九七九年全面检查处理结束，总结上报并附报清查成果表，包括各级清查数字。
