中共上海市委宣传部办公室《思想动向》141

日期：1965-11-30 作者：中共上海市委宣传部办公室编印

中共上海市委宣传部办公室编印

本市演出、出版有关海瑞的戏、书和评介文章目录

据本部办公室同志初步了解，自一九五七年以来，本市演出、出版有关海瑞的戏、书和评介文章，编目于后：

（一）本市剧团演出有关海瑞的剧目：

《海瑞上疏》，上海京剧院，集体创作，许思言（原名许铁生）执笔。

导演：周信芳。副导演：李桐森、马科。演于一九五九年、一九六一年、一九六二年。

《海瑞》。黄浦京剧团，集体创作。陈水钟执笔。

导演周正邦、演于一九五九年八月——一九六零年、一九六一年、一九六三年。

《海瑞》，出新剧团，创作：陈忠衍、张小林、吕茂洪等。导演：张小林、吕茂洪。知道；薛传？。演于一九五九年一月。

《海瑞罢官》。烽火准剧团。移植，导演：程少楠，演于一九六三年九月。

《碰粮船》，人民评弹团，整理，演于一九六一年。

《海瑞认女》，人民评弹团。整理，演于一九六一年。

《大红袍》，人民评弹团，传统老书。演于一九六一年以前。

《大红袍》，长征评弹团，传统老书，演于一九六一年以前，程鸿？主演。

《大红袍》，长征评弹团，传统老书，演于一九六一年以前，浦？峰主演。

《海瑞上疏》，凌霄评弹团。改编，演于一九六一年十月。

（二）本市出版社出版有海瑞书籍和连环画：

《海瑞上疏》（京剧）。上海京剧院许思言编剧，陶雄写“前言”，上海文化出版社，一九六零年。

《海瑞》（京剧），黄浦京剧团集体创作，陈水钟执笔，上海文化出版社出版。

《海瑞》，作者：蒋星煜。上海人民出版社，一九五七年九月出版。

《海瑞》，连环画，按蒋星煜原作改编，改编者：黎帜昌。绘画者：常焕文。上海人民美术出版社，一九六一年十一月出版。

《海瑞的故事》作者：蒋星煜。少年儿童出版社，一九六零年四月出版。

《海瑞》。（彩色连环画故事）编文：布谷。绘画：仲达。发表于《儿童时代》一九五九年十七期。

（三）本市报刊发表有关海瑞的文章：

（1）评介海瑞戏的文章：

《杂谈京剧<海瑞上疏>》，作者：卫明（原名张丙昆。上海京剧院编剧）。解放日报，一九六一年二月十一日发表。

《试评周信芳的新作<海瑞上疏>》。作者：卫明。交汇报，一九五九年十月二十七日发表。

《评京剧<海瑞上本>》，作者：朱素君。新闻日报，一九五九年九月二十九日发表。

《评京剧<海瑞上疏>》，作者：徐以礼（中国唱片厂上海编辑分社主任），上海戏剧，一九五九年十一期刊载。

《谈<海瑞奏本>》，作者：梁伟初。上海戏剧，一九五九年十一期刊载。

《<海瑞上本>中的海瑞形象》，作者：小捷。解放日报，一九五九年九月二十六日发表。

《“南包公”海瑞》，作者：吕仲（上海京剧院编剧），劳动报，一九五九年五月六日发表。

《包公与海瑞》，作者：闻亦步（文汇报文艺部）。文汇报，一九五九年五月十三日发表。

（2）有关海瑞故事的文章：

《南包公——海瑞》，作者：蒋星煜（市文化局干部）。解放日报，一九五九年四月十七日发表。

《海瑞》，程十发画，布谷文，交汇报，一九五九年五月二十——六月三日连载。

《海瑞和真理》，作者：冯俊。新闻日报，一九五九年六月三日发表。

《海瑞的故事》，布谷文，赵宏本（人美连环画创作室党员副主任）图，劳动报，一九五九年五月九日——二十三日连载。

《海瑞和吴淞江》，作者：华龙，新民晚报，一九五九年五月二十七日发表。

《海瑞奏本》，作者：华龙。新民晚报，一九五九年八月十七日发表。

《一生节俭的海瑞》，作者：布谷，新民晚报，一九五九年九月六日发表。

《海瑞与何以尚》。作者：布谷，新民晚报，一九五九年十月二日发表。

《南包公海瑞》，作者：蒋星煜，少年文艺，一九五九年八月号刊载。

来源：上海档案馆档案号 A22-2-1280-108
