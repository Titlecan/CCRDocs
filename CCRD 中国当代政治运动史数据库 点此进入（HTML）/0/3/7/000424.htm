周恩来关於上海工总司成立一周年庆祝活动的指示

日期：1967-11-14 作者：周恩来

周恩来

（大意）

上海工总司成立一周年的庆祝活动不要搞得大，可以开一些小型的庆祝会，要大搞特搞反修。

来源：1967年11月15日清华大学井冈山斗批改战团编印《学习资料》第43期。
