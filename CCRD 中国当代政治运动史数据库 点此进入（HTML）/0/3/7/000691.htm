阎长贵给清华大学井冈山总部的电话

日期：1967-2-5 作者：阎长贵

阎长贵

我前几天来你校看了一下大字报，发现了你们反“托派”，这是不对的，不应该把矛头指向群众，而应当指向当权派，一般同学有错误缺点应当批评帮助，反托派对本身不好，对团结也不好，请转告蒯大富。

来源：

1967年3月北京玻璃总厂红卫兵联络站编《中央首长讲话（2）》
