杨成武关锋关于批斗李曼村谢镗忠的意见

日期：1967-6-0 作者：杨成武, 关锋, 张秀川

杨成武、关锋、张秀川

《誓把军队无产阶级文化大革命进行到底》联络站代表和部分海军造反派代表在三座门揪斗李曼村、谢镗忠。海军政治部主任张秀川同志传达杨代总长、关锋同志的九点意见，内容大意如下：

1. 李、谢可以给你们斗。

2. 萧华要做检查。

3. 彻底改组全军文革院校组。

4. 检查、清理院校组整造反派的黑材料。

来源：

中国科学院化学所铁锤战斗队编《动态》第115期，1967年6月19日。
