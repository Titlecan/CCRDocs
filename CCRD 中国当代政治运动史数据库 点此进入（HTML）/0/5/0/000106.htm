红卫兵战歌（歌词）

日期：1966-8-1 作者：清华大学附属中学红卫兵

清华大学附属中学红卫兵

先辈血战创江山，革命后代来接班。跟着领袖毛泽东，造反小将千千万。誓死保卫党中央，誓死保卫毛主席，大风大浪英勇奋战，高歌猛进杀向前。

纵横南北红卫兵，牛鬼蛇神心胆寒。毛泽东思想红战旗，世世代代永相传。造反！造反！造反！敢叫日月换新天！杀！

红卫兵创作1966.8.1
