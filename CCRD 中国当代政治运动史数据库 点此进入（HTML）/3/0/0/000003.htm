中共中央情报部关于解决土地问题的方式给东北局的指示

日期：1946-5-19 作者：[待确定]

（一）关于木兰、通河、方正三县群众工作经验很好，我们已转给热河作为他们的参考。

（二）经验中证明必须使清算运动发展到分配开拓地、满拓地，这是很正确的，但根据太行、山东、华中的经验，清算运动不仅可以用来清算敌伪及汉奸土地，而且可以用清算方式解决地主阶级的土地。即利用清算租息，清算额外剥削（如大斗进小斗出等），清算无偿劳役，清算转嫁担负（如应由地主担负之地亩捐，转嫁于农民担负），清算霸占吞蚀，清算人权污辱（如地主强奸和霸占农民妻女等）等种种方式，使地主土地在偿还积债、交纳罚款、退还霸占、赔偿损失等合法名义下，转移到农民手里和折算或出卖到农民手里。此种经验，望你们注意。

（三）望你们通告各地迅速地放手解决满洲土地问题，以巩固我党在满洲的群众基础。
