全面结束土地改革的工作既须做完又须做好

日期：1952-9-0 作者：《长江日报》社论

《长江日报》社论

中共中央中南局召开的农村工作会议指出：当前农村正处在由改革到建设的过渡时期。会议确定全面作好结束土地改革的工作，为今冬明春农村三大工作任务之一。会议曾经强调指出：全面结束土地改革运动，争取一个彻底全面的胜利，就可以为今后长期的经济建设任务，奠定一个长远的基础；也就是说，把旧的历史任务完成得好，才能使新的历史任务在一个胜利的基础上开始。中南局农村工作会议的精神和确定的方针，无疑是完全正确的。

正由于当前是处在从改革到建设的过渡时期，时间十分紧迫，在有限的时间内，要完成这样一个巨大的工作，任务是非常艰巨的；因此，在工作中必须争取时间，务须于明年春耕前全面地完满地结束这一工作，以便迎接祖国大规模的经济建设；否则，将不适应全国的建设任务，而陷于被动。

今冬明春，农村战线上面临这样一个紧迫而又艰巨的任务，就要求各级领导同志，再接再厉，集中指导，提高方法，务须使土地改革的结束工作做到既好且快。

但是，贯彻与执行这样一条正确的方针，障碍还是存在的。目前有若干同志，对这一方针表示怀疑，对于全面结束土地改革运动的重要意义认识不足；或者由于土地改革已近尾声，自然地产生一种涣散情绪，似乎不主张有一个土地改革的结束阶段，至少认为不必要像过去三年来指导土地改革运动那样认真了。

这种观点是有害的。历史经验告诉我们，任何改革运动都须有其酝酿阶段、大发展阶段和巩固阶段。三年来的土地改革运动，我们经过了酝酿阶段与大发展阶段，取得了土地改革运动的基本胜利。但是除少数地区外，巩固阶段的工作，还未认真的开始，有赖于今冬明春补完这个重要的一课。

土地改革之所以还须要一个巩固阶段，就是因为大发展阶段是一个运动的主潮，是一个革命飞跃；运动像暴风雨一样的猛烈迅速，往往是几个月的时间就完成一个大革命；正因为运动进展迅速，就不可避免的有粗糙现象。敌人被基本上打倒了，但残余的敌人尚未被肃清；打倒了的敌人，还来不及分别发落，在运动中错打了自己人的，也还没有来得及做补救工作；特别是群众的思想觉悟程度和组织程度，还有一定限制，在若干地区还不足以巩固住已得的胜利。因此，如果没有一个结束阶段，把革命进行彻底，对遗留问题完满的加以解决，不但不能发展胜利和巩固胜利，就是已得的成果也不可免地要回生一部分，而这些遗留问题，又将在今后建设任务中起着消极作用和破坏作用。

从实际情况看，更能说明这个问题。据中南土地改革委员会所进行的100个乡调查的结果，群众发动不平衡不充分的现象，还是严重的存在着。大约有20％至30％的乡，还处于夹生或半夹生状态，那里的农民大多数还没有获得政治上的优势与经济上的适当满足，地主阶级当权派还未打倒，基本上还保持着旧的封建秩序。另外，在大约70％的乡，地主作为一个阶级已被废除了。其中80％的地主，在政治、经济各方面遭到沉重的打击；但现在还有20％左右的地主，仍有违法活动。农民的土地问题是基本上解决了；但其中还有30％左右的雇贫农，对于土地的分配状况表示不满。这些人恰是生产战线上的一部分主力；他们对土地尚未获得适当满足。这个问题并非客观上无法解决，而是可能解决却没有解决好。这就说明，过去三年来的土地改革运动，成绩虽然是很大的，但仍有需要发展的方面；我们估计已得的成绩的时候，必须清醒地估计到这一点。同时各地复查试点的经验也证明：即使在农民已基本取得优势的乡，也还普遍存在着许多遗留问题，如山林、水利尚未分配、土地证尚未颁发，斗争果实被积压或虽已分配但分配不公，特别是错划阶级，扩大了打击面，以及基层干部不够坚强、干部和群众的关系不够协调等；这些问题不加以适当解决，就会脱离群众，不但影响农民内部团结，也会影响农民对发展生产的情绪。其次，对地主尚未分别发落，而且普遍存在着管制面太宽、太滥等现象；这是极不策略的。必须在珍贵群众热情的情况下，教育广大农民群众对敌人实行区别对待；应根据地主违法、半守法、守法等不同政治态度，分类排队，区别处理，打击和从严管制少数不法分子，从宽处理守法分子，并予以生活出路，以达分化敌人，消灭敌人之目的。这是巩固农村人民民主专政不可缺少的步骤，也必须在巩固阶段中，加以解决。总之，不论从历史经验和当前农村实际情况看来，都说明了任何革命运动，于大踏步前进之后，必须要一个巩固阶段来收场；这是整个运动不可逾越的一个环节。那种“土改结束阶段可有可无”的论调，是毫无根据的。

另外还有些同志认为，广大农民群众迫切地要求生产，似乎不再要求复查了，这种说法也是不完全合乎实际情况的。他们只看到问题的一方面，而没有看到另一方面。的确，生产是农民的迫切要求之一，为此，中南局农村工作会议，将组织群众生产作为今冬明春三大工作任务之一，并强调在结束土地改革运动当中，坚持从生产入手，不误生产，最后归结到生产的原则。因为土地改革的直接目的，就是为了解放农村生产力，发展农业生产。土地改革之后，农民劳动在自己的土地上所业已表现出来的生产积极性，正是近三年来土地改革运动的成果；但是否能够从此得出结论说农民只要生产而不要把土地改革运动进行到底呢？当然不能这样说。事实是最有说服力的。各地材料都说明，在一切夹生和半夹生的乡村，农民的第一个要求仍然是政治上的解放和经济上的适当满足，要求获得生活资料和生产资料，以便取得发展生产的条件。在农民业已取得优势的乡村，处理遗留问题，如纠正错划阶级，改善干部群众之间的关系，分别处理地主，巩固人民民主专政等，都是群众的迫切要求；至于确定地权颁发土地证，更是广大农民普遍的迫切要求。这些遗留问题不适当的加以解决，就不可能有个很稳定的生产关系，各阶层人民的生产积极性，就不能够充分发挥出来。因此，那些将结束土地改革运动与发展生产对立起来的看法是不对的。恰恰相反，我们认真地结束土地改革，正是为发展生产打基础，正是直接服务于生产运动的。因为我们所要求的生产运动是建立在一种新的生产关系上的生产运动，是建立在广大农民群众高度革命热情基础上的一种生产运动；没有这样的一个基础，就不会真正出现一个广大群众性的爱国丰产运动。明白了这个道理之后，就可以确定的说：把结束土地改革运动与生产运动对立起来的说法，同样是没有根据的。

还有一种议论，说什么：“为了换取时间，争取主动，对于土地改革结束阶段只好马虎一点”。是的，我们应该争取主动，并且要学会争取主动，这是做领导工作最重要的艺术，问题是如何去争取主动。企图以省略反封建的最后一役，不顾千百万农民群众的要求和热情，来换取时间争取主动的想法是错误的而且也不可能达到目的。在目前全国即将进入大规模经济建设的新形势下，时间是十分宝贵的。我们要争取时间，在今冬明春有限的时间内，认真地踏踏实实地将结束土地改革的工作做完做好，从这种意义上来掌握争取主动，是完全正确和必要的。道理很简单，一切决定于群众，一切事业都是千百万群众的革命行动，能否争取主动的关键，在于群众是否充分的发动。充分地发动和组织群众的结果，乃是今后建设任务中长期起作用的因素。如果我们能够团结和依靠农民群众的大多数，并使他们的思想觉悟和组织程度获得进一步的提高，那就是我们取之不尽用之不竭的力量源泉，就可立于主动的地位。我们要求今冬明春认真做好结束土地改革的工作，正是为了这样的目的。反之，如果采取草率从事单纯赶任务的做法，那不是在争取主动，而是自己在制造被动。任何对于结束土地改革的工作采取偷工减料、敷衍塞责的做法，都将使我们今后一切建设工作被迫地在“松松垮垮”的基础上去进行，终将陷全盘工作于长期的被动。因此，企图对于土地改革的结束阶段采取草率态度，以换取时间争取主动的说法，也是毫无根据的。

目前是处在由改革到建设的最后过渡时期，新旧任务交错，而且非常繁重。正是由于处在这样一个历史转折关头，越要求各级领导同志坚持不懈，继续努力，不乱步骤，坚决贯彻全面结束土地改革的任务；这是打基础的工作，是百年大计。迎接国家建设任务的各种准备工作中，最为重要的是准备群众条件；群众的革命热情是克服一切困难的保证。如果由于沉不住气，放松了这个中心环节，而单纯地去“赶任务”，势必“欲速则不达”，将来在建设工作中将以双倍代价来做弥补工作。目前土地改革工作正是“功亏一篑”的时候，望各地同志百尺竿头更进一步。
