反对和资产阶级斗争中的右倾观点

日期：1952-2-22 作者：西北局《党内通讯》第93期社论

西北局《党内通讯》第93期社论

在毛主席和党中央号召全党把反贪污、反浪费、反官僚主义的斗争，看作如同镇压反革命斗争一样的重要，如同镇压反革命一样的发动群众，大张旗鼓、雷厉风行地去进行的时候，只有一部分领导同志，尚能根据运动发展情况，在毛主席的指示下，思想认识逐步明确，并且提出办法，推动运动前进，而有许多领导同志，却表现了惊人的政治麻痹，异乎寻常的感觉迟钝，他们的领导思想一直落后于实际。在反贪污、反浪费、反官僚主义运动开始的时候，他们总自以为是地认为自己领导的地区和工作部门“没有问题”或者“问题不大”，“钱少、地方穷，那里有严重的贪污浪费呢？”当他们在“限期完成、首长包干”的严厉督责下，被迫走上战场，稍有一点成绩时，就认为解决了问题，准备收兵。当运动进到需要集中力量，反对大贪污分子的时候，他们又喊着“山中无虎”或者表示“没有把握”，对资产阶级安在我党我军内部的堡垒，侵吞人民国家财富的大小“老虎”，不去寻找，不去大胆怀疑，信心不高，气魄不大，制订打“虎”计划，总是十分保守。接受“打虎”任务，总是面有难色，劲头不足，战志不强，松懈拖拉，满不在乎，推一推动一动，极不自觉。试问在这粉碎资产阶级进攻的严肃的阶级斗争中，采取如此麻痹的态度，表现如此的懦弱无能，这不是右倾表现是什么？在和资产阶级作斗争中，所以发生上述那些右倾思想，固然不能否认，有其客观原因，但是对于我们共产党人来说，尤其是对党的领导者来说，并不能把责任完全推给客观原因，如果这样就不能得出有益的教训。因为，毛主席和党中央早就警告过我们，中国的民族资产阶级是具有两重性的阶级，“如果我们党不知道在联合资产阶级时又同资产阶级进行坚决的、严肃的‘和平’斗争，党在思想上、政治上、组织上就会瓦解，革命就会失败。”（见“共产党人”发刊词）也早就指出中国革命在全国胜利之后，国内的基本矛盾“即无产阶级与资产阶级的矛盾”（见二中全会决议），要我们预防敌人的“糖衣炮弹”的攻击。而毛主席和党中央在最近以来，则更是电如星火，不断向我们提醒资产阶级进攻的严重危险，指出开展“三反”斗争的重要意义。不仅如此，骇人听闻的严重事实每天都在暴露着，那一个地区、那一个部门，没有发现贪污浪费现象？没有发现资产阶级进攻的大小事件？就不提过去，只说现在。为什么在毛主席和党中央的警告下，在铁一样的事实面前，还未觉悟起来，引起应有注意呢？说我们缺乏和资产阶级进行这种“和平”斗争的经验吧，这固然是事实。因为我们过去主要是和帝国主义封建势力进行斗争的，和资产阶级作斗争确实是生疏的；但是真正的斗争经验，都是在进行斗争中摸索出来的，“不入虎穴，焉得虎子”，为什么不去摸呢？而目前党内电报、报纸刊物上，都大量刊载着、传播着这类经验，为什么又不去吸取和学习呢？

那么，究竟问题在那里呢？没有别的，就是这些同志自己已经沾染了资产阶级的灰尘，受了资产阶级的感染。进入城市后，许多人讲究“大方”、“排场”、“比较待遇”，党内滋长了骄傲情绪、居功思想、不求进步、贪图享乐。有的表现为脱离实际、脱离群众，缺乏和本阶级群众的紧密连系，缺乏对资产阶级——这个敌对阶级情况的系统调查研究；因而敌人在暗害我们，我们却熟视无睹。把一个无产阶级的政治家，变成了一个庸庸碌碌的官僚主义者。有的发展了资产阶级单纯的技术观点与纯业务观点，严重的忽视政治，忽视阶级斗争。此外，有少数人发生了严重的浪费和贪污行为，以至变成了资产阶级的走卒，成了党与无产阶级的叛徒，人民的罪人。既然沾染了程度不一、表现不同的资产阶级思想情绪，那么在反对资产阶级的斗争中，就会程度不一地蒙蔽着对客观真理的认识，减弱对资产阶级思想侵蚀的警惕，降低了与资产阶级进行斗争的决心与劲头，这是很明白的，不容争辩的。

但是，我们许多共产党员和党的领导者，在进入城市之后，为什么恰恰接受了资产阶级的思想影响，而没有增强工人阶级的思想意识呢？在城市中，存在着的并不只是资产阶级，还有和这个阶级相对立的工人阶级。资产阶级是我们的敌对阶级，它先天就带着堕落腐化，工人阶级是我们的本阶级，只有它才是最先进的伟大的。我们共产党员——工人阶级的先锋队，为什么不去联系工人阶级，加强和巩固工人阶级的思想意识，却反而眼睛向着资产阶级，接受了资产阶级的影响呢？回答这个问题，不能用资产阶级进攻去解释，因为资产阶级总是力图影响我们的，要向我们进攻的，正如我们不能希望老虎不吃人一样。必须从这里找寻答案，就是由于我们许多同志出身于小资产阶级，长时期处在和小生产者接触的工作环境里，还或长或短地拖着小资产阶级的尾巴，还保有苦干小生产者的思想情绪；而这种思想情绪是和资产阶级思想相亲近的，是和资产阶级思想相投合的，是资产阶级思想体系热心的自发的拥护者，是跟着资产阶级走并向资产阶级地位上爬的。保存着小生产者散漫、动摇的本色，以及其自私自利、个人打算的思想，再加上马克思、列宁主义理论水平低下，思想意识修养不够，不仅不能警惕和抵御资产阶级的思想侵袭，而且会不自觉的接受或看中于资产阶级的生活方式；不能和资产阶级作斗争并战胜之；而表示懦弱无能，麻木不仁，就是很自然的，也是被事实所证明了的。

领导者首先把自己从资产阶级思想影响下解救出来，这是我们加强和资产阶级进行斗争的基本前提。因为任何受了资产阶级影响、中了资产阶级污毒的人，决不可能坚决的、胜利的领导反资产阶级的严肃斗争。毛主席和党中央反复强调着的“首长负责，层层检讨”的方针，就是要求各级党的领导同志首先深刻检讨自己，彻底洗清自己，然后才好提高和改造别人，领导群众进行斗争。因此，目前我们各级党的领导同志，必须再仔细读一读毛主席和党中央历次的重要指示，深思熟虑，体会其精神与实质，认真看一看反贪污、反浪费、反官僚主义运动中暴露出来的资产阶级猖狂进攻的恶毒事实，认清资产阶级的丑恶面目，它的进攻的危险性；冷静检讨自己在进入城市后的思想作风与生活习惯，特别是在反贪污、反浪费、反官僚主义运动中的表现和态度，并虚心听取下边群众的批评与意见，以坚定和增强自己的无产阶级的意识，严格的和小资产阶级、资产阶级思想划清界限。我们共产党员，特别是党的领导，只有扫除了自己身上资产阶级的灰尘，加强了自己的无产阶级的阶级感情与义愤，在和资产阶级作斗争中才会意志坚定，毫不动摇，一身是胆，力大无穷；才会耳聪目明、智慧百出，不管敌人如何隐蔽，如何狡猾，都会依靠群众，想出方法，搜索净尽，获得全胜；也再不会是政治上的庸人，而是坚定、勇敢、明智的政治活动家。在具有伟大意义的反贪污、反浪费、反官僚主义运动中，在这个和资产阶级的严肃斗争中，党员尤其党的领导者，向这个方向努力锻炼吧！
