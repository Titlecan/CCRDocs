无例外地有领导地立即彻底检查与分配积压的斗争果实

日期：1952-5-20 作者：《中南“农村工作》社论

《中南“农村工作》社论

反对积压群众斗争果实不分的偏向，虽已强调多次，但至今还有些同志误认为这种偏向并不严重，因而不去认真彻查和迅予分配。其实并非如此，前些时河南省深入检查发现，尚有经过土地改革业已为时2年（即在1949年冬季和1950年春季进行土地改革）的乡村，至今还有积压群众斗争果实未分；最近湖北江陵县检查了25个业已结束土地改革的乡村，发现积压了群众斗争果实谷6,169石（平均每乡有24,676斤谷）未分；天门县深入检查，发现黄潭区熊河乡积压7,000万元和田庙乡积压1,200万元的果实粮，均存在银行未分，并已被乡干部贪污浪费一部，该区朱店乡，有百余石田未分下去，是该乡农民集体种的，一部分棉花桃子和粮食还放在房里未分；广东省检查，发现竟有积压群众斗争果实不分，坐视群众春荒困难不能解决而发生饿死人现象。

请加注意：上述的事例是发生在业已结束土地改革而非正在结束的乡村（有的已是二年之久）；是大批而非少数的积压群众斗争果实不分；是发生在群众生产上仍有不少困难的今天；是业已发生了贻误群众的生产和群众的渡荒；并且业已发挥出腐蚀着干部的坏作用。同时，这种积压群众斗争果实不分的偏向，决不只是河南、湖北、广东的问题而是普遍的、严重的存在着。

不论正在进行土地改革与复查的地区，或业已结束土地改革与复查的地区，均必须足够重视这一问题。应该算个账，应该心中有数。一乡之内积压上千万元（甚至是几千万元）的斗争果实，在你领导的地区，是有多么大的一个生产资金呀！立即分配给雇贫农是对雇贫农的生产、生活有多大帮助呀！再不要只是在那里空喊“扶助雇贫”，“满足雇贫”的口号了，应该行动起来，从各方面的、具体的实际行动中来“扶助雇贫”，“满足雇贫”。

各地应立即专门讨论与处置积压斗争果实不分的问题。领导干部应亲自动手，抓紧指导，贯彻始终。应教育每个工作同志，如果再不改变或重犯这种不珍惜群众斗争果实，对群众利益、死活不负责任的恶劣作风，就是有意犯错。应该指令每个乡村必须认真检查与彻底分配。不论是什么斗争果实，均必须在这次检查与处理中，予以彻底分配，并必须首先满足雇贫农。处理积压斗争果实的工作与当前生产运动并不矛盾，而且正是为了有益于当前生产运动，因此必须无例外的在每个乡村有领导的开展一次彻底检查与分配积压群众斗争果实的工作。
