坚决打击地主阶级“假分田”的阴谋活动

《中南土改简报》短论

湖北沔阳、公安、松滋、潜江等县，所发现地主阶级“假分田”的阴谋活动，实值各地高度警惕与严加注视。此种事件，乃是地主阶级抵抗、破坏土地改革最毒辣的一种手段和伎俩。其恶果不仅是地主阶级混水摸鱼，黑白不分，破坏政策；更严重的是地主阶级以此来麻醉群众，与我争夺群众，遂其“夜渡陈仓”混过关去，照旧维持封建统治秩序。地主阶级所以胆敢如此，是乘隙于我工作同志的自满麻痹；群众的分田心切和农村基层不纯。

在春耕生产季节日益迫切的情况下，地主阶级将更会“趁火打劫”制造此种“假分田”事件。因此，各地必须立即警惕起来，给地主阶级此种阴谋伎俩以坚决打击。

应重申和认真执行“中南区惩治不法地主暂行条例”第六条之规定：“以不法行为假冒农会组织进行分配土地者”，“处以三年以上五年以下有期徒刑”。不论何处发生此种事件，皆应立即迅予处理，严惩主谋者，扩大宣传，以警效尤。绝不允许地主阶级的罪行蔓延。

必须在尚未进行分配土地地区的农民中，揭发地主阶级这一阴谋诡计；要耐心的向农民群众进行说服解释工作，告诉雇贫农：土地改革是一定要进行的，田地是一定要进行分配的，共产党与人民政府是坚决帮助雇贫农翻身的；不过，所希望农民的，是农民不仅自己要组织起来，而且要在共产党与人民政府的领导下，要在彻底打倒地主阶级的情况下，来分配土地，以达永享“国泰民安”的日子，不要只看到目前可以分田而不去彻底打倒地主阶级，留下“后患无穷”。以免农民群众误中敌计。至于农民本年的生产困难问题，可以告诉群众，当地人民政府和农民协会是会帮助解决的。

应该通告周知，人民政府不允许自发分田，更不允许乡村干部随意行事。不论何乡分田，必须有人民政府和上级的农民协会派人去领导才能进行。如有故意如此肇事者，人民政府一定查究。

我所有从事土地改革工作的同志，必须由此事件，提高自己的敌情观念，更加深入地去了解情况，识破地主阶级的真面貌；更应努力地去深入发动群众并领导群众深入的进行斗争。切不可再忽视敌情，更不可再急性赶活，以免为敌所乘。
