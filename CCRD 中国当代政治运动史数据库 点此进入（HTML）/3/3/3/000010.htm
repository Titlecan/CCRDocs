克服右倾思想穷追贪污犯

《东北日报》社论

我们反贪污、反浪费、反官僚主义斗争，进入穷追大贪污犯阶段，已经二十多天了。在这二十多天中间，斗争形势发展得非常迅速，不少单位追捕大贪污犯的成绩，与日俱增。这些单位所以能得到这种成绩，主要是领导思想明确，认识了大贪污犯是吞噬国家财产的“大老虎”，追捕大贪污犯要像去年镇压反革命那样，充分发挥群众威力，加强调查研究，从各方面搜集材料，寻找线索，把群众威力和调查研究材料结合起来，使贪污犯绝无藏身余地，一一落网。但是，还有些单位，对穷追大贪污犯的斗争表现得软弱无力，时间一天天过去，或者毫无成绩，或者是成绩很小。为什么会有这种情况，主要是领导上存在着右倾思想。这种右倾思想表现在以下几个方面:

第一种右倾思想表现在:明明是大贪污犯在自己身边钻来钻去，而他竟以为自己的单位没有“老虎”。这种右倾思想是阻碍反贪污斗争深入的最危险的思想。具有这种思想的人，往往找出各种各样的理由来进行自慰。有的说:“我们这里不管大宗钱财物资，是‘清水衙门’绝不会有大贪污犯。”有的说:“我们这里虽然经手钱财物资，但是我们的各种手续制度都很严格，没有漏洞可钻，小贪污犯难免存在，大贪污犯一定没有。”有的说:“我们机关建立很久，过去经过几次整顿，有些贪污分子均经过处分，再不会有什么大贪污犯的。”有的说:“我们机关建立不久，不会有大贪污犯。”最严重的，竟有人当上级指示其努力追捕大贪污犯并给以奋斗目标之后，不但不竭尽全力去积极进行这一严重斗争，完成这一光荣的任务，反而发牢骚，抱怨上级是“不了解情况，主观主义地提出要求。”这种右倾思想已被日益深入的斗争，和从深入斗争中捕捉大批大贪污犯的事实，完全证明了是十分错误的。首先这种右倾思想是对三年来的资产阶级的猖狂进攻熟视无睹。资产阶级用“拉”和“派”的办法，在我们队伍里，无论党政军民学哪个系统都包括在内，安插了许多代理人和走狗，即各种各样的贪污分子。有右倾思想的人，只看到贪污分子是穿的革命工作人员衣服，而就错把他们当成同志看待，却不认识他们的危害国家和人民的真实面貌。因而也就没有决心和勇气向这些贪污分子进行坚决的斗争。其次，随着反贪污反浪费反官僚主义斗争的深入，许多过去认为没有贪污分子的单位，现在事实证明有很多大小贪污犯存在。经管大批财物的单位丢开不说，就是所谓“清水衙门”的单位，经管财物较少的政法文教系统，也发现了“大老虎”。这种事实难道还不应该引起有右倾思想的领导人猛醒，而深入地去检查自己所属的单位吗。有这种右倾思想的人，应当马上端正自己的思想，充分认清情况，下定决心，发动群众，勇敢作战，坚决向大贪污犯进攻，猛打穷追，除恶务尽，不要再犹豫了。

第二种右倾思想表现在:对贪污犯特别是大贪污犯的狡猾估计不足，轻易相信他们的话。有的贪污犯说:“我保证自己没有贪污，如果查出来，愿受加倍处分。”这话明明是一种“以守为攻”的战术。而有右倾思想的人，竟然相信不疑。有的贪污犯，知道自己罪恶已经暴露，不可能一口咬定没贪污，于是采取“以退为进”的办法，或者承认小的掩盖大的，或者承认过去的隐瞒现在的，或者承认现在的隐瞒过去的，或者承认一件隐瞒其他，花样很多，万分狡猾。有右倾思想的人，满足于已经追查出的轻微贪污事实，就完全听信了贪污犯的谎话，而再不深追。有的贪污犯卖“老资格”，要赖皮，夸功劳，求可怜。而有右倾思想的人，也就大发慈悲之心，以这种或那种的理由，对他们姑息起来。他不懂得对盗窃国家财产的贪污犯的仁慈，就是对人民的最大残忍；也不懂得如果不严肃地穷追和惩处那些顽固不化的贪污犯，就不能挽救和改造那些愿意悔过自新的贪污犯。

第三种右倾思想表现在:追查贪污犯，不是根据人证物证定案，而是认为必须贪污犯本人承认，如不承认，即不敢认定他是贪污犯。不错，不少一时失足的贪污犯，一经号召，立即自动坦白讲出过去贪污事实。这种贪污犯，往往是罪恶不大，或者是中毒不深的人。也有一些贪污犯，对于人民政府宽大政策缺乏认识，对自动坦白存在许多思想顾虑，经过反交代政策，而后也能承认自己的罪恶。但是，还应当看到，也有一批贪污犯顽固不化的。他们只有继续贪污作恶之心，绝无悔改自新之意。他们想以“拖”和“混”的办法，度过这次反贪污反浪费反官僚主义的斗争，妄想一阵大风刮过去，将来还可以照旧为非作歹。因此，他们建立起什么“攻守同盟”，互相约定“掉头不谈，攻破谁算谁”；用尽心机地修筑几道所谓防线，第一道是矢口不讲，第二道是避重就轻，第三道是要死狗混时间，还有……。他们有一句

话，叫做“宁肯死在刀下，不能死在话下。”对于这种最顽固、最凶恶的贪污犯，绝不能和那些可以争取改造的贪污犯同样看待。如果希望他们亲口认罪，岂不是等于劝说老虎不要伤人一样幼稚可笑吗?我们是马克思列宁主义者，处理一切案件，完全根据事实，不凭口供。凡是有了人证物证的贪污犯，即可定案，不管他本人是否承认。同时，凡是事实俱在而自己矢口不承认者，一定要给以加重处分。

第四种右倾思想表现在:自己不去积极创造追捕贪污犯的各种条件，而是坐待时机成熟，彷彿贪污犯特别是大贪污犯，都是有政治自觉的人，用不着化费甚么力气，只要一经号召，他们就会自动投案。有这种右倾思想的人，表面上也很积极，不住口地喊叫“捉老虎”，上午一个大会进行号召，下午一个大会进行动员，但是除了说空话之外，再拿不出别的办法。其结果是群众的斗争积极性被浪费掉了，而“老虎”仍是逍遥法外。这种情况绝不应再继续下去了，必须马上加以改变，采取有效办法，组织坚强的指挥部和包干打虎队，调查研究各方面的材料，进一步加强和扩大群众的“打虎”威力，从各方面创造“打虎”的条件，以便迅速获得“战果”，彻底查出一切大小贪污犯。

我们追查大贪污犯的成绩正在猛烈增长，这方面的经验也在日益丰富，从全般情况来看，对大贪污犯的上下夹攻和内外夹攻的形势已经形成，彻底穷追一切贪污犯的条件已经具备了。只要我们能以彻底地克服掉各种右倾思想，则我们再经过一番紧张的努力，一定可以获得全部胜利。
