向伟大斯大林祝寿的献礼

日期：1949-12-21 作者：《人民日报》社论

《人民日报》社论

全世界觉悟了的人民，今天都以无限的热情，庆祝全人类革命的导师，伟大斯大林的七十大寿。获得了革命胜利的觉悟了的中国人民，今天更是狂热地来庆祝他，这是因为斯大林对中国人民革命事业的帮助太大了。

中国人民自从一百年前中国社会沦为半封建半殖民地的时候起，就发生了反帝国主义的革命要求。当时中国人民的革命要求是以当时的中国社会和与之相适应的物质生活条件及其相当的政治制度为前提的。在当时中国社会的生活条件下，微弱的资产阶级刚才开始反对封建专制主义的斗争；当时中国的无产阶级正处在生长的初期，也还不能有直接实现自己的阶级利益的企图。由于中国社会内部的阶级斗争在当时那样的不发展状态，就使得中国在鸦片战争以后的八十年间，所有大大小小的革命运动，都没有能够找到正确的社会科学或社会规律的知识为指导，而相继失败了。只有随着社会生活条件的变化，在中国逐渐生长的无产阶级开始自觉地形成为一个阶级之后，又遇到伟大的俄国十月革命的胜利，这时候，才使中国人能够找到马克思列宁主义，很快觉悟到要走俄国人的路；这时候才有了中国共产党；这时候，中国的革命才进入了一个崭新的历史时期。而从这个时期刚一开始，伟大的斯大林就一直关心着中国的革命，在理论上和事业上日益增强他对于中国人民和中国革命的援助。

斯大林在对中国革命问题上，和在对其他问题上一样，充分表现了他发挥马克思列宁主义的极大功劳。马克思在他生活的年代，曾预见世界上最古老的中国“处于社会革新的前夜”，他指出“这种革新对于世界文明，无论如何应有非常重大的结果”。列宁在生前进一步指出对于中国这样的半殖民地的国家，应该“最坚决地援助这些国家内资产阶级民主的民族解放运动中最革命的分子，帮助他们的起义，在发生革命战争的时候，就帮助他们的革命战争，去反对压迫他们的帝国主义列强”。列宁还具体地帮助了孙中山先生的革命事业。马克思和列宁对中国革命最早的分析，是永远不会被人忘怀的，同样，我们更会永远记得斯大林是继马克思、列宁之后，第一个光辉地系统地全面地解决中国革命的基本问题并不断给予中国人民以大量实际援助的伟大领袖、导师和朋友。斯大林不仅明确地分析了中国革命的反帝反封建的性质，并且他根据中国社会的民族的特点，在他的著作中说明了中国革命的根本路线和革命统一战线等主要策略原则，指出中国革命是以武装斗争为主要形式和农民问题的重要性。斯大林在中国革命问题上所作的马列主义的理论贡献是极为重大极为富的。为了保卫与贯彻在中国革命问题上的马列主义正确路线，斯大林曾进行了一系列最严肃的理论斗争，粉碎了托洛斯基派对中国问题的一切无耻谎言。现在事实已经充分证明：每当极端复杂曲折的中国革命临到紧急关头的时候，以毛泽东同志为首的中国共产党人的思想，无论在党的建设原则上，在军事思想上，在统一战线思想上，都是和伟大斯大林的思想相吻合，因而它是经得起严重考验的完全正确与无敌的。试想：如果没有按照列宁、斯大林党的建设原则而建立起来的布尔塞维克的中国共产党来领导中国的革命，如果没有中国共产党所领导的中国人民解放军的英勇善战，如果没有中国共产党所倡导的以工人阶级为领导、以工农联盟为基础、包括各民主党派、各人民团体、各民主阶级、各民族及其他爱国民主分子的广泛的人民民主的革命统一战线，那么，中国人民反对帝国主义、反对封建主义、反对官僚资本主义的革命怎样能够取得今天这样伟大的胜利呢？显然的，只有在伟大的斯大林和毛泽东完全一致的对于中国革命的正确思想指导之下，中国革命才能取得今天的胜利。这不仅说明了毛泽东思想是马列主义的普遍真理和中国革命的具体实践相结合的科学理论，说明了毛泽东同志是斯大林最好的学生和战友，而且更说明了斯大林对中国革命问题的科学分析和预见的精深与伟大。这个科学思想在中国已经变成巨大的物质力量，它已经在中国生根，在中国开花，在中国结果了。由此可见，中国人民所以会热烈爱戴与感谢斯大林是决非偶然的。

中国人民之所以会热爱与感谢斯大林，还由于他和他所领导的苏联政府和人民，始终一贯地给予中国人民以平等的友爱的无条件的援助，使中国人民从实际体验中能够认识这空前未有的最可靠的忠诚无私的朋友。当中国人民处在黑暗和反动压迫的年月，当中国人民的斗争处境艰难与危险的时候，斯大林和他所领导的苏联政府和人民，总是用了一切可能的努力，在道义上、精神上和物质上给予中国人民以必要的援助。远的不说，即以中国抗日战争时期而论，当美英帝国主义还在资助与鼓励日寇侵略的时候，苏联首先援助中国抗战。前后四次共达三亿美元的贷款、数百余贺飞机及大批军火之外，还有大批空军人员。许多苏联的英雄儿女的鲜血，曾飘洒在中国的长空。在张鼓峰、诺蒙坎两次战役中，苏联红军更直接打击了日寇，并继续牵制了日寇精锐的关东军，直到最后予以残灭，逼使日寇投降。在外交上，苏联曾极力主张集体制裁日寇，扩大援华，它动员了一切舆论的力量，始终支援中国人民的伟大抗战直到胜利。最近，当我中华人民共和国成立之后，苏联不但首先与我国建立外交关系，而且很快从各方面给予我国以真诚友谊的帮助。所有和苏联朋友接触过的人们都深深地被他们的国际主义的无私和忘我的服务精神所感动，这是因为苏联人民，正如刘少奇同志所说的，是“在伟大的列宁和斯大林的教育培养下的人民，这种人民对我们中国人民以至全世界的人民是具有无限忠诚的国际主义的友爱的。他们对中国人民的帮助是无条件的，不要求报酬的。”“因为世界上的共产党人，我们中国的共产党人也在内，是世界上一种完全新的人物，这种新的人物是具有无限真诚的国际主义友爱精神的，能够无条件地、无分别地为世界任何地方的人民服务。”斯大林就是世界上这种新的人物的最杰出的代表，因此他成为全世界进步人类的导师，受到全人类最崇高的尊敬与爱戴。

今天我们中国人民热烈庆祝斯大林的七十大寿，最重要的就是要努力学习斯大林，学习斯大林所教育培养下的苏联人民。这应该是我们向伟大斯大林祝寿时最隆重最珍贵的献礼。因为斯大林的最可宝贵和最可尊敬的地方，就在于他的伟大的思想。斯大林的思想即是科学的马列主义。在中国，它的具体实现即是毛泽东思想。中国人民在一个世纪以来，特别在四运动以来，曾不惜一切牺牲，努力追求民主和科学。现在我们不但有了人民民主即新民主主义的社会现实，而且更有了全人类最进步的科学——马列主义的无尽宝藏。今后的中国，必须有更多的人来学习和掌握这个科学，才能够巩固中国革命所已得的胜利，才能够争取在祖国的伟大建设事业中继续获得更多的胜利，才能够取得全人类解放事业的胜利。有了这个科学，我们就能够得到一切。让我们在伟大的斯大林和毛泽东所代表的马列主义的革命的科学的旗帜下，努力前进！
